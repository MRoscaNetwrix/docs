# SMARTLog:Â Target Log for Windows Event Log Type

The Target Log page is where logs are selected to be collected. This version is a wizard page for the log type of Windows Event Log.

![SMART Log DC Wizard Target Log page for Windows Event Log](../../../../../../../../static/images/AccessAnalyzer_12.0/Content/Resources/Images/EnterpriseAuditor/Admin/DataCollector/SMARTLog/TargetLogWindowsEvent.png "SMART Log DC Wizard Target Log page for Windows Event Log")

Only one log can be targeted per query task. The selected log is displayed at the bottom of the wizard page.
