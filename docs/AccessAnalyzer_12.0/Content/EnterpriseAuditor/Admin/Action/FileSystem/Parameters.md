# File System Action: Parameters

The Parameters page is available for some of the selections on the Operation page. The list of operations below provides access to the operation-specific versions of the Parameters page for this wizard. Click on an operation to view its associated Parameters page.

* [Change Attributes](Parameters/ChangeAttributes "Change Attributes")
* [Change Permissions and Auditing](Parameters/ChangePermissionsAuditing "Change Permissions and Auditing")
* [Change Permission Inheritance](Parameters/ChangePermissionInheritance "Change Permission Inheritance")
* [Change Share Permissions](Parameters/ChangeSharePermissions "Change Share Permission")
* [Remove File Permissions](Parameters/RemoveFilePermissions "Remove File Permissions")
* [Remove Share Permissions](Parameters/RemoveSharePermissions "Remove Share Permissions")
* [Add Tags](Parameters/AddTags "Add Tags")
* [Remove Tags](Parameters/RemoveTags "Remove Tags")
* [Change Owner](Parameters/ChangeOwner "Change Owner")

![File System Action Module Wizard Change File Attributes Parameters page](../../../../../../../static/images/AccessAnalyzer_12.0/Content/Resources/Images/EnterpriseAuditor/Admin/Action/FileSystem/Parameters/ChangeAttributes.png "File System Action Module Wizard Change File Attributes Parameters page")

The Navigation pane will list this as the Parameters page, but the title for each version indicates the type of parameter to be configured.
