# 3. Quarantine Job Group

The 3. Quarantine job group is designed to report on and quarantine files that are pending cleanup.

![3. Quarantine Job Group in the Jobs Tree](../../../../../../../../static/images/AccessAnalyzer_12.0/Content/Resources/Images/EnterpriseAuditor/Solutions/FileSystem/Cleanup/Quarantine/JobsTree.png "3. Quarantine Job Group in the Jobs Tree")

This job group includes the following jobs:

* [FS\_QuarantineData Job](FS_QuarantineData "FS_QuarantineData Job") – Designed to quarantine files subject to be cleaned up
* [FS\_QuarantineData\_Status Job](FS_QuarantineData_Status "FS_QuarantineData_Status Job") – Designed to report on the FS\_QuarantineData job
* [FS\_RestoreInheritance Job](FS_RestoreInheritance "FS_RestoreInheritance Job") – Designed to restore inheritance to previously quarantined files
* [FS\_RestoreInheritance\_Status Job](FS_RestoreInheritance_Status "FS_RestoreInheritance_Status Job") – Designed to report on inheritance that was restored to previously quarantined files

Workflow

The following is the recommended workflow for using the job group:

**Step 1 –** **Prerequisite:** Successfully execute the following job groups:

* .Active Directory Inventory
* File System > 0.Collection
* File System > Cleanup > 1. Cleanup Assessment

**Step 2 –** Schedule the 3. Quarantine job group to run as desired after the prerequisites have completed.

**Step 3 –** Review the reports generated by the 3. Quarantine job group.
