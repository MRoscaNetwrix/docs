# PgSQL\_Configuration Job

The PgSQL\_Configuration Job is designed to return additional configuration settings from PostgreSQL servers.

## Queries for the PgSQL\_Configuration Job

The PgSQL\_Configuration Job uses the SQL Data Collector.

**CAUTION:** Do not modify the query. The query is preconfigured for this job.

![Query Selection](../../../../../../../../static/images/AccessAnalyzer_12.0/Content/Resources/Images/EnterpriseAuditor/Solutions/Databases/PostgreSQL/ConfigurationQuery.png "Query Selection")

The query is:

* PostgreSQL Database Sizing - Collects details about PostgreSQL databases. See the [SQL Data Collector](../../../../Admin/DataCollector/SQL/Overview "SQL Data Collector") topic for additional information.
