# 8.S3 Sensitive Data Job Group

The 8.S3 Sensitive Data job group provides details on AWS S3 buckets and objects containing sensitive data.

![8.S3 Sensitive Data Job Group in the Jobs Tree](../../../../../../../static/images/AccessAnalyzer_12.0/Content/Resources/Images/EnterpriseAuditor/Solutions/AWS/SensitiveData/JobsTree.png "8.S3 Sensitive Data Job Group in the Jobs Tree")

The 8.S3 Sensitive Data job group is comprised of:

* [AWS\_SensitiveData Job](AWS_SensitiveData "AWS_SensitiveData Job") – Provides details on AWS S3 buckets and the objects in them which contain sensitive data
* [AWS\_SensitiveData\_Permissions Job](AWS_SensitiveData_Permissions "AWS_SensitiveData_Permissions Job") – Provides details on the permissions assigned to AWS S3 buckets and the objects in them which contain sensitive data
