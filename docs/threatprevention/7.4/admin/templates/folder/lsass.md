# LSASS Folder Templates

The **Templates** > **Microsoft** > **LSASS** folder contains the following templates:

| Template                                 | Description                                                                                 | TAGS |
| ---------------------------------------- | ------------------------------------------------------------------------------------------- | ---- |
| LSASS Guardian <ul><li>Monitor</li></ul> | No customizations required. Detects attempts by other processes to alter the LSASS process  | None |
| LSASS Guardian <ul><li>Protect</li></ul> | No customizations required. Prevents attempts by other processes to alter the LSASS process | None |
