# Select AD Groups Window

The Select AD Groups window provides a list of available Active Directory groups. Selected objects
are added to the filter from where you clicked the **Add** (+) button to open this window.

![Select AD Groups Window](/images/threatprevention/7.4/admin/policies/eventtype/window/selectactivedirectory/adgroups.webp)

- Select either Browse Mode or Search Mode.

- Select an Agent from the drop-down menu and click **Connect**.
- Navigate through the domain tree and check the desired item(s) in the **Results** pane on the
  right.
- Then click **OK** to close the window.

See the
[Selection Windows](/docs/threatprevention/7.4/admin/policies/configuration/eventtype/window/overview.md)
topic for additional information.

The selection is displayed in the appropriate box of the AD Account filter or the AD Groups filter.
