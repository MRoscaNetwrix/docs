# Delete User

The Administration Console prevents you from deleting the last user with Administrator rights. There must always be at least one active administrator.

Follow the steps to delete a user.

__Step 1 窶点_ Click __Configuration__ > __Users__ on the menu to open the Users and Roles window.

![Remove User button on the Users and Groups window](/static/img/product_docs/threatprevention/threatprevention/admin/configuration/userroles/removeuser.png)

__Step 2 窶点_ Select a user and click the __Remove__ (x) button in the upper-right corner to delete it.

__Step 3 窶点_ _(Optional)_ Delete as many users as desired before clicking OK.

__Step 4 窶点_ Click __OK__ to save changes on this window.

The deleted user(s) have no access to the Administration Console.
