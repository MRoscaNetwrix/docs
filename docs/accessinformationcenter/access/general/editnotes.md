# Edit Notes Window

The Edit Note window can be opened from a variety of interfaces. Follow the steps to add or edit a note.

__Step 1 –__ Select the item in the interface and click Edit Notes. The Edit Notes window opens.

![Edit Notes window showing note entry field](/static/img/product_docs/accessinformationcenter/access/general/editnotes.png)

__Step 2 –__ Type or edit the note in the textbox.

__Step 3 –__ Click OK when finished. The Edit Notes window closes.

The user name and a date timestamp will appear at the beginning of each note added.
