# Access Request Reminder Email

The Request Administrator may send reminder email from the Access Information Center for pending access requests.

![Reminder email](/img/product_docs/accessinformationcenter/access/informationcenter/accessrequests/email/reminder.webp)

Click __Sign in__ to open the Access Information Center login page. Within the Owner portal, navigate to the Access Requests page to process the request. See the [Pending Access Requests](/docs/accessinformationcenter/access/informationcenter/accessrequests/owners/pendingrequests.md) topic for additional information.
