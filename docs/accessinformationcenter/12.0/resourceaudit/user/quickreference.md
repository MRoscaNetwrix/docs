# User Reports Quick Reference

The following reports are available for selection within the User Audit interface for an Active
Directory user:

| Report                                              | Description                                                                                                                                                                                                                                                    |
| --------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Activity Details Report](/docs/accessinformationcenter/12.0/resourceaudit/user/activitydetails.md)       | Provides details on every activity event logged by the audited user during the selected date range. This report includes a Permission Changes table.                                                                                                           |
| [Activity Statistics Report](/docs/accessinformationcenter/12.0/resourceaudit/user/activitystatistics.md) | Provides statistical activity event information for the audited user during the selected date range. This report includes a line graph for Traffic Trend.                                                                                                      |
| [Attribute Changes Report](/docs/accessinformationcenter/12.0/resourceaudit/user/attributechanges.md)     | Provides specific details for every attribute change to the audited user during the selected date range.                                                                                                                                                       |
| [Effective Access Report](/docs/accessinformationcenter/12.0/resourceaudit/user/effectiveaccess.md)       | Provides insight into every resource the audited user has access to and what level of access has been granted. Effective access is a calculation based on several variables according to the type of resource. This report includes a Permission Source table. |
| [Member Of Report](/docs/accessinformationcenter/12.0/resourceaudit/user/memberof.md)                     | Provides a list of all groups of which the audited user is a member. This report includes a Membership Paths table.                                                                                                                                            |
| [Object Permissions Report](/docs/accessinformationcenter/12.0/resourceaudit/user/objectpermissions.md)   | Provides details on Active Directory permissions for the object.                                                                                                                                                                                               |
| [Permissions Report](/docs/accessinformationcenter/12.0/resourceaudit/user/permissions.md)                | Provides a list of all resources where the audited user has been assigned permissions.                                                                                                                                                                         |
