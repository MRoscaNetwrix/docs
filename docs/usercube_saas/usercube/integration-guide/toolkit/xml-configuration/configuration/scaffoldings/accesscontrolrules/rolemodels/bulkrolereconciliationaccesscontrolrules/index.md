# Bulk Role Reconciliation Access Control Rules

Generates the permissions to perform bulk validations on the __Role Reconciliation__ page.

The scaffolding generates the following scaffoldings:

- [
  Reconciliate Roles Access Control Rules
  ](../reconciliaterolesaccesscontrolrules/index.md): Generates the permissions to access the role reconciliation pages for a given entity type and profile.

## Properties

| Property | Details |
| --- | --- |
| EntityType   required | __Type__    String   __Description__   Identifier of the entity type involved in the scaffolding. |
| Profile   required | __Type__    String   __Description__   Identifier of the profile involved in the scaffolding. |
