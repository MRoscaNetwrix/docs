# Notification: Summary

The Summary Page displays all the information input in each of the configured options from the previous pages of the wizard.

![Notification Data Analysis Module wizard Summary page](/static/img/product_docs/accessanalyzer/accessanalyzer/enterpriseauditor/admin/datacollector/adinventory/summary.png)

Click __Finish__ to save configuration changes. If no changes were made, it is best practice to click __Cancel__ to close the Notification Data Analysis Module wizard to ensure no accidental clicks are saved.
