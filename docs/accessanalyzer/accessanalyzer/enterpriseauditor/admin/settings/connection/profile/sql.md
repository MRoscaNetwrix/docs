# SQL Authentication for User Credentials

This information applies to __Select Account Type__ > __SQL Authentication__ in the User Credentials window.

__NOTE:__ SQL Authentication credentials are used in the Connection Profiles for the SQL, MySQL, and PostgreSQL Solutions.

![User Credentials - SQL Authentication](/img/product_docs/accessanalyzer/accessanalyzer/enterpriseauditor/admin/settings/connection/profile/sqlauthentication.png)

The required credentials for SQL Authentication are:

- User name – Enter user name
- Password Storage – Application (Uses the configured Profile Security setting as selected at the __Settings__ > __Application__ node. See the [Application](/docs/accessanalyzer/accessanalyzer/enterpriseauditor/admin/settings/application/overview.md) topic for additional information.)
- Password – Type the password
- Confirm – Re-type the password
