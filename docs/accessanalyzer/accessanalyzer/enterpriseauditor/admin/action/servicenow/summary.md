# ServiceNow Action: Summary

The Summary page displays a summary of the configured query.

![ServiceNow Action Module wizard Summary page](/static/img/product_docs/accessanalyzer/accessanalyzer/enterpriseauditor/admin/datacollector/adinventory/summary.png)

Click __Finish__ to save configuration changes. If no changes were made, it is a best practice to click __Cancel__ to close the ServiceNow Action Module Wizard to ensure that no accidental clicks are saved.
