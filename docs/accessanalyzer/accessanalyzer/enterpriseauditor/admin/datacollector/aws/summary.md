# AWS: Summary

The Summary page displays a summary of the configured query. It is a wizard page for all categories.

![summary](/static/img/product_docs/accessanalyzer/accessanalyzer/enterpriseauditor/admin/datacollector/adinventory/summary.png)

Click __Finish__ to save configuration changes. If no changes were made, it is a best practice to click __Cancel__ to close the Amazon Web Services Data Collector Wizard to ensure that no accidental clicks are saved.
