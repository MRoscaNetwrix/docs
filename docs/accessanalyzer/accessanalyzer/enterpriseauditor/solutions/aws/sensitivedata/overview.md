# 8.S3 Sensitive Data Job Group

The 8.S3 Sensitive Data job group provides details on AWS S3 buckets and objects containing sensitive data.

![8.S3 Sensitive Data Job Group in the Jobs Tree](/static/img/product_docs/accessanalyzer/accessanalyzer/enterpriseauditor/admin/hostmanagement/jobstree.png)

The 8.S3 Sensitive Data job group is comprised of:

- [AWS\_SensitiveData Job](/docs/product_docs/accessanalyzer/accessanalyzer/enterpriseauditor/solutions/aws/sensitivedata/aws_sensitivedata.md) – Provides details on AWS S3 buckets and the objects in them which contain sensitive data
- [AWS\_SensitiveData\_Permissions Job](/docs/product_docs/accessanalyzer/accessanalyzer/enterpriseauditor/solutions/aws/sensitivedata/aws_sensitivedata_permissions.md) – Provides details on the permissions assigned to AWS S3 buckets and the objects in them which contain sensitive data
