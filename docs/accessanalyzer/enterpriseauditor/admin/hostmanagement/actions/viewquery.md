# View Query

Use the __View Query__ option to open the Host Discovery Queries pane.

![View Query option on Activities pane](/img/product_docs/accessanalyzer/enterpriseauditor/admin/hostmanagement/actions/viewquery.png)

Click __View Query__ to go to the Host Discovery Queries pane. See the [Host Discovery Queries](/docs/accessanalyzer/enterpriseauditor/admin/hostdiscovery/queries.md) topic for additional information.
