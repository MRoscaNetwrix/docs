# 4.Roles Job Group

The 4.Roles job group provides details on roles in the AWS IAM environment.

![4.Roles Job Group in the Jobs Tree](/img/product_docs/accessanalyzer/enterpriseauditor/admin/hostmanagement/jobstree.png)

The 4.Roles job group is comprised of:

- [AWS\_Roles Job](/docs/accessanalyzer/enterpriseauditor/solutions/aws/roles/aws_roles.md) – Provides details on roles in the AWS IAM environment
- [AWS\_StaleRoles Job](/docs/accessanalyzer/enterpriseauditor/solutions/aws/roles/aws_staleroles.md) – Provides details on roles that are considered stale. Highlighting roles that have not been used in more than 60 days and those that have never been used.
