# Create Active Directory User Objects

In an Active Directory identity stores, you can create the following types of users:

* User – See the [Create an AD User](User "Create an AD User") topic for additional information.
* Mailbox – See the [Create an AD Mailbox](Mailbox/Mailbox "Create an AD Mailbox") topic for additional information.
* Contact – See the [Create an AD Contact](Contact/Contact "Create an AD Contact") topic for additional information.
