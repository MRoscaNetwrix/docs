# Delete a Workflow Request

Delete a workflow request from a specified identity store ID of which is specified in the endpoint URL.

## Endpoint

https://machinename:4443/GroupIDDataService/api/IdentityStores/{identitystoreId}/Workflows/{WorkflowRequestID}

## HTTP Methods

DELETE

#### Sample Response Syntax

```
true
```
