# Add Taxonomy

On this step, you are prompted to load predefined taxonomies.

![processing_settings_taxonomies](/img/product_docs/dataclassification/ndc/install/initialconfiguration/processing_settings_taxonomies.png)

Click the search bar and select one or several taxonomies you want to add. See [Built-in Taxonomies Overview
](/docs/dataclassification/ndc/admin/taxonomies/builtintaxonomies.md) for the full list of built-in taxonomies supported by Netwrix Data Classification.
