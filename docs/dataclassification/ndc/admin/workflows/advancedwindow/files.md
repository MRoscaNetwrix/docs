# Advanced Actions for File System

In addition to the __Email Alert__ ,__Migrate Document__ and additional classification, the following actions are available for the __File system__ content source type:

- __Update Permissions__ — this action updates the file system permissions for the classified document. See [Update Permissions](/docs/dataclassification/ndc/admin/workflows/actions/updatepermissions.md) for details.
- __Apply MIP Label__, __Remove MIP Label__ — these actions, respectively, apply and remove sensitivity label to/from a document stored on a file system, using Microsoft Information Protection (MIP). This helps to automate protection policies application. See Modify MIP Label for details.

To configure actions for file systems using the Advanced interface:

1. In administrative web console, navigate to Workflows and select the workflow you need.
2. Click the workflow, then click Add next to Rule Actions.
3. In the Add Action dialog, select the action you need from the __File System__ section in the Action Type list.
