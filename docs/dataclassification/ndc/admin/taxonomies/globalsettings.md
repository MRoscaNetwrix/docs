# Global Settings

The __Content__ > __Taxonomies__ > __Global Settings__ panel allows you to update, compare, back up, export, edit or delete your taxonomies. See the [Manage Taxonomies](/docs/product_docs/dataclassification/ndc/admin/taxonomies/manage.md) topic for additional information.
