The table below lists types of content and their default extensions supported out of the box.

__NOTE:__ To review the full list of available content types, navigate to Config → Text Processing → Content Type Extraction Methods.

| --- | --- |
| .aiff | AIFF |
| .bmp | Bitmap |
| .chm | Compiled HTML |
| .doc | Word |
| .docx | Word Xml |
| .dwg | CAD |
| .eml | Exchange Mail |
| .flv | FLV |
| .html | HTML |
| .java | Java Source |
| .jpg | JPEG |
| .mpp | Project |
| .msg | Message |
| .pdf | PDF |
| .webp | png |
| .ppt | Powerpoint |
| .pptx | Powerpoint Xml |
| .pub | Publisher |
| .rar | Archive |
| .rtf | Rich Text |
| .tiff | Tiff |
| .tmp | Unknown |
| .txt | Text |
| .vsd | Visio |
| .vtl | Dictionary / VTL |
| .wav | WAV |
| .wp | Word Perfect |
| .xls | Excel |
| .xlsx | Excel Xml |
| .xml | XML |
| .zip | Archive |
| .7z | Archive |
