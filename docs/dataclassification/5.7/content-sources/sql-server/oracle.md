# Oracle Server

Once you choose to add SQLÂ server source, you opt to add Oracle server as a source. For more
information, go to the main article[SQL Server ](/docs/dataclassification/5.7/content-sources/sql-server/index.md).

To connect to an Oracle server:

1. On the **Source Configuration** screen, select **Oracle**.
2. Specify Oracle server, username, Container DB Service Name, and Oracle password.

   ![sqlserver_source](/img/product_docs/dataclassification/ndc/admin/sources/sqlserver/sqlserver_source.webp)

3. Select Save.
