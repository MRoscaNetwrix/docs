# Deployment Checklist

This section contains the pre-installation checklist, installation notes or observations, and
post-installation tasks. Review these steps to plan and deploy Netwrix Data Classification:

|                                         |                                                                                                                                                         |                                                                                                                 |
| --------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------- |
| Prepare evaluation server and test data |                                                                                                                                                         |                                                                                                                 |
| 1                                       | Prepare your test data set and ensure it is accessible over the network from the Netwrix evaluation server using your test account.                     | - [Supported Data Sources](/docs/dataclassification/5.7/getting-started/requirements/supported-content.md)      |
| 2                                       | Make sure your evaluation server is suitable for running all the Netwrix Data Classification components including a standalone SQL Server instance for. | - [Hardware Requirements](/docs/dataclassification/5.7/getting-started/requirements/hardware.md)                |
| 3                                       | Make sure your evaluation account meets security requirements.                                                                                          | - [Accounts and Required Permissions](/docs/dataclassification/5.7/getting-started/requirements/accounts.md)    |
| 4                                       | Deploy prerequisite Windows features and additional software.                                                                                           | - [Software Requirements](/docs/dataclassification/5.7/getting-started/requirements/software.md)                |
| Deploy Netwrix Data Classification      |                                                                                                                                                         |                                                                                                                 |
| 5                                       | Prepare IT infrastructure.                                                                                                                              | - [Configure IT Infrastructure](/docs/dataclassification/5.7/configuration/infrastructure/index.md)             |
| 6                                       | Install Netwrix Data Classification.                                                                                                                    | - [Install Netwrix Data Classification](/docs/dataclassification/5.7/installation/index.md)                     |
| Configure Netwrix Data Classification   |                                                                                                                                                         |                                                                                                                 |
| 7                                       | Provide a name for new instance                                                                                                                         | - [Initial Product Configuration](/docs/dataclassification/5.7/installation/initial-configuration/index.md)     |
| 8                                       | Select processing (indexing) mode.                                                                                                                      | - [Select Processing Mode](/docs/dataclassification/5.7/installation/initial-configuration/modes.md)            |
| 9                                       | Configure global processing settings.                                                                                                                   | - [Processing Settings](/docs/dataclassification/5.7/installation/initial-configuration/processing-settings.md) |
| 9                                       | Add built-in taxonomies.                                                                                                                                | - [Add a Taxonomy](/docs/dataclassification/5.7/taxonomies/create-taxonomy.md)                                  |
| 10                                      | Review your configuration.                                                                                                                              | - [Review Your Configuration](/docs/dataclassification/5.7/installation/initial-configuration/review-config.md) |
| 11                                      | Configure NDC SQL database.                                                                                                                             | - [Configure NDC SQL database](/docs/dataclassification/5.7/getting-started/requirements/databases.md)          |
| 12                                      | Add a source.                                                                                                                                           | - [Add a Content Source](/docs/dataclassification/5.7/content-sources/add-source.md)                            |
| Review classification results           |                                                                                                                                                         |                                                                                                                 |
| 13                                      | Start by reviewing the Document Tagging report and browsing classification results.                                                                     | - [Review Reports and Browse Classified Documents](/docs/dataclassification/5.7/reporting/review-reports.md)    |
