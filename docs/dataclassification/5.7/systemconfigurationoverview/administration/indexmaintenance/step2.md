# Step 2: Maintenance Options

Specify options for the operation you have selected.

| Operation selected | Available options                                                                                                        | Details                                                                                                                                                                                                                 |
| ------------------ | ------------------------------------------------------------------------------------------------------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Rebuild Index**  | **Shrink the "text.cse" file?** <ul><li>**Shrink**</li><li>**Don't Shrink** (default)</li></ul>                          | Selecting **Shrink** will rebuild the _Text.cse_ file, removing any fragmentation. **Shrink** will require sufficient disk space to process (up to the existing size of _Text.cse_) and may take some time to complete. |
| All operations     | **Would you like to re-run the product configuration wizard?** <ul><li>**Run**</li><li>**Don't Run** (default)</li></ul> | Select **Run** if you want to re-configure this instance by going through the initial steps of the product configuration. Note that this will pause all sources.                                                        |
