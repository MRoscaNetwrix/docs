# Global Settings

The **Content** > **Taxonomies** > **Global Settings** panel allows you to update, compare, back up,
export, edit or delete your taxonomies. See the [Manage Taxonomies](/docs/dataclassification/5.7/taxonomies/manage-taxonomies.md) topic for additional
information.
