# Add Taxonomy

On this step, you are prompted to load predefined taxonomies.

![processing_settings_taxonomies](/images/dataclassification/5.7/install/initialconfiguration/processing_settings_taxonomies.webp)

Click the search bar and select one or several taxonomies you want to add. See
[Built-in Taxonomies Overview ](/docs/dataclassification/5.7/contentconfigurationoverview/taxonomies/builtintaxonomies.md) for the full list of
built-in taxonomies supported by Netwrix Data Classification.
