# Accounts and Required Permissions

Netwrix Data Classification uses the following accounts:

| Account             | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| ------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Service Account** | This account is specified during the product setup. Windows domain account that you plan to use as a service account will need the following: <ul><li>Local Administrator rights on the server where Netwrix Data Classification will be installed.</li><li>Permissions to run the Windows Services and IIS Application pool.</li><li>SQL Server DBO permissions to the NDC SQL database (if using Windows Authentication to access SQL Server). After installation, this account will be automatically granted the **Logon as a service privilege** on the Netwrix Data Classification server.</li></ul>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| Crawl content       | Ensure the availability of accounts with sufficient permissions to access your content sources: <ul><li>SharePoint, SharePoint Online site collection— Site Collection Administrator role.</li><li>Exchange mailboxes: 1. **ApplicationImpersonation** —allows the crawling account to impersonate each of the mailboxes / users configured for collection. 2. **Mailbox Search** —allows the crawling account to enumerate mailboxes, i.e. automatic discovery of mailboxes. See [Configure Microsoft Exchange for Crawling and Classification](/docs/dataclassification/5.7/introduction/introduction/exchange/exchange.md) for detailed information on configuring these permissions.</li><li>Outlook Mail Archive (PST file)— **Read** permission.</li><li>File System (SMB, NFS) — **Read** permission for the folders and files you need to crawl.</li><li>G Suite and Google Drive —service account needs permissions to read data in the individual and shared Drives on behalf of users using the Google Drive API. See [Configure G Suite and Google Drive for Crawling](/docs/dataclassification/5.7/introduction/introduction/gdrive.md) for detailed information.</li><li>Database— **Read** permission for the database schema and data.</li></ul> |
| Apply tagging       | To use tagging, i.e. to write classification attributes back to the content file, crawling account will need the appropriate **Modify** permissions on the content source.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
