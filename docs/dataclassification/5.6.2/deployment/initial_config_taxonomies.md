# Add Taxonomy

On this step, you are prompted to load predefined taxonomies.

![processing_settings_taxonomies](/img/product_docs/dataclassification/5.6.2/deployment/processing_settings_taxonomies.webp)

Click the search bar and select one or several taxonomies you want to add. See
[Built-in Taxonomies Overview ](/docs/dataclassification/5.6.2/taxonomies/standalone_taxonomies.md)
for the full list of built-in taxonomies supported by Netwrix Data Classification.
