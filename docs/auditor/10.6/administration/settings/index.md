# Netwrix Auditor Settings

In the Settings section, you can configure product settings, such as default SQL Server instance for
Audit Database, the Long-Term Archive location and retention period, etc. You can also review
information about the product version and your licenses. See the following sections:

- [General](/docs/auditor/10.6/administration/settings/general-settings.md)
- [Audit Database](/docs/auditor/10.6/administration/settings/database-settings.md)
- [Long-Term Archive](/docs/auditor/10.6/administration/settings/archive-settings.md)
- [Investigations](/docs/auditor/10.6/administration/settings/investigation-settings.md)
- [Notifications](/docs/auditor/10.6/administration/settings/notification-settings.md)
- [Integrations](/docs/auditor/10.6/administration/settings/integration-settings.md)
- [Licenses](/docs/auditor/10.6/administration/user-management/licensing.md)
- [About Netwrix Auditor](/docs/auditor/10.6/administration/settings/general-settings.md)

To modify Netwrix Auditor settings, you must be assigned the _Global administrator_ role. See
[Role-Based Access and Delegation](/docs/auditor/10.6/monitoring-plans/delegation.md)
for more information.
