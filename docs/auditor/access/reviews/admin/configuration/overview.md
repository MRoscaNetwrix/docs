# Configuration Interface Overview

The Configuration interface is available only to users with the Administrator role. It is opened by the __Configuration__ tab.

![Configuration interface showing the Console Access page](../../../../../../static/img/product_docs/accessanalyzer/enterpriseauditor/admin/settings/access/rolebased/consoleaccess.webp)

It has the following pages:

- [Console Access Page](consoleaccess.md) – Grant users console access
- [Active Directory Page](activedirectory.md) – Configure the Active Directory service account used to add console users.
- [Notifications Page](notifications.md) – Configure the SMTP server, email security settings, notification options, and owner reminder settings
- [Database Page](database.md) – Configure the connection to the database
- [Diagnostics Page](diagnostics.md) – Download logs and enable debug log level for troubleshooting
