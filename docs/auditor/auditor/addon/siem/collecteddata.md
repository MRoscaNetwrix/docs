# Work with Collected Data

Follow the steps to work with collected data:

__Step 1 –__ On the computer where you executed the add-on, navigate to __Start__ > __All Programs__ > __Event Viewer__.

__Step 2 –__ In the Event Viewer dialog, navigate to __Event Viewer (local)__ > __Applications and Services Logs__ >Netwrix Auditor Integration log.

__Step 3 –__ Review events.

[![EventLog_Export_Example](/static/img/product_docs/auditor/auditor/addon/solarwinds/eventlogexportexample_thumb_0_0.png)](/docs/product_docs/auditor/resources/images/auditor/addon/eventlogexportexample.png)

Now you can augment Windows event log with data collected by the Auditor.
