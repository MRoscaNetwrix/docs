# Work with Collected Data

Follow the steps to see collected data.

__Step 1 窶点_ Log on to your ArcSight Logger web interface.

__Step 2 窶点_ On the __Summary__ page, select the __Event Summary by Receiver__ diagram and click the __UDP Receiver__ segment (Activity Records are imported through UDP Receiver). Select __TCP Receiver__ if you specified TCP protocol for transferring data.

__Step 3 窶点_ On the __Analyze__ page that opens, review the search field. Ensure your computer is listed as Receiver (e.g., "_172.28.156.131 [UDP Receiver]_"). If you imported Activity Records from more than one Netwrix Auditor Server, add all of them in the search field.

__NOTE:__ You might want to modify time range and the fields shown.

![activityrecords](/img/product_docs/auditor/auditor/addon/arcsight/activityrecords.webp)

__Step 4 窶点_ Review imported Activity Records.
