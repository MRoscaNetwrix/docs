# Netwrix Auditor Settings

In the Settings section, you can configure product settings, such as default SQL Server instance for Audit Database, the Long-Term Archive location and retention period, etc. You can also review information about the product version and your licenses. See the following sections:

- [General](/docs/product_docs/auditor/auditor/admin/settings/general.md)
- [Audit Database](/docs/product_docs/auditor/auditor/admin/settings/auditdatabase.md)
- [Long-Term Archive](/docs/product_docs/auditor/auditor/admin/settings/longtermarchive.md)
- [Investigations](/docs/product_docs/auditor/auditor/admin/settings/investigations.md)
- [Notifications](/docs/product_docs/auditor/auditor/admin/settings/notifications.md)
- [Integrations](/docs/product_docs/auditor/auditor/admin/settings/integrations.md)
- [Licenses](/docs/product_docs/auditor/auditor/admin/settings/licenses.md)
- [About Netwrix Auditor](/docs/product_docs/auditor/auditor/admin/settings/about.md)

To modify Netwrix Auditor settings, you must be assigned the _Global administrator_ role. See [Role-Based Access and Delegation](/docs/product_docs/auditor/auditor/admin/monitoringplans/delegation.md) for more information.
