# Reporting

The Reporting module is used to generate basic reports in CSV using [API query grammar](/docs/product_docs/usercube/usercube/integration-guide/api/squery/index.md), or advanced reports using the [
Business Intelligence
](/docs/product_docs/usercube/usercube/integration-guide/toolkit/xml-configuration/business-intelligence/index.md) module.

See the [
Generate Reports
](/docs/product_docs/usercube/usercube/user-guide/administrate/reporting/index.md) topic for additional information on generating reports.
