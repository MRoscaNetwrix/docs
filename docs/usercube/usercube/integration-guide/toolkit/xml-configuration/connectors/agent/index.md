# Agent

Contains all the running agents.

## Properties

| Property | Details |
| --- | --- |
| DisplayName\_L1   required | __Type__    String   __Description__   Display name of the agent in language 1 (up to 16). |
| Identifier   required | __Type__    String   __Description__   Agent Identifier. |
| State   default value: 0 | __Type__    Int32   __Description__   Agent Status ("Unknown"=0, "Online"=1 and "Offline"=2). |
| URI   optional | __Type__    String   __Description__   Agent URI. |
