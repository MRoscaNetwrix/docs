# Installation Guide

This guide is designed to help you install Identity Manager in a production environment.

## Target Audience

This guide is intended for __system administrators__ and __system architects__.

Required knowledge includes:

- Windows Server administration
- Internet Information Services (IIS) administration
- SQL Server administration

## Overview

The installation of Identity Manager requires architectural decisions to be made. An [
Overview
](/docs/usercube/usercube/installation-guide/overview/index.md) of the architecture and available configurations will help you make informed decisions.
