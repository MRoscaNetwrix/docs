    deploy them.- ####
    [Create Menu Items](/versioned_docs/usercube_6.1/usercube/integration-guide/ui/how-tos/create-menu-items/index.md)
    Create the menu items required to access the workflow in the UI.- ####
    [Customize Display Tables](/versioned_docs/usercube_6.1/usercube/integration-guide/ui/how-tos/custom-display-table/index.md)
    Define a custom way to display entity types' data.- ####
    [Customize Search Bars](/versioned_docs/usercube_6.1/usercube/integration-guide/ui/how-tos/custom-search-bar/index.md)
    Define a custom way to search from a list of a given entity type's properties.- ####
    [Customize Forms](/versioned_docs/usercube_6.1/usercube/integration-guide/ui/how-tos/custom-forms/index.md)
    Define a custom way to display the input fields to be filled in a given workflow.
