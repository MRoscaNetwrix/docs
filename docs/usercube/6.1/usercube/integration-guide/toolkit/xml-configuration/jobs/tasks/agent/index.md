# Agent Tasks

- #### [ActivityInstanceActorTask](/docs/usercube/6.1/usercube/integration-guide/toolkit/xml-configuration/jobs/tasks/agent/activityinstanceactortask/index.md)
    Update the Actors for the workflows instances- ####
    [CreateDatabaseViewsTask](/docs/usercube/6.1/usercube/integration-guide/toolkit/xml-configuration/jobs/tasks/agent/createdatabaseviewstask/index.md)
    Generates entity model SQL views in the Usercube database.- ####
    [ExportTask](/docs/usercube/6.1/usercube/integration-guide/toolkit/xml-configuration/jobs/tasks/agent/exporttask/index.md)
    Runs the specified connection's export.- ####
    [FulfillTask](/docs/usercube/6.1/usercube/integration-guide/toolkit/xml-configuration/jobs/tasks/agent/fulfilltask/index.md)
    Retrieves provisioning orders from the informed connector generated by
    GenerateProvisioningOrdersTask to make changes in a system. Instead of a connector it is
    possible to launch it with a list of TaskResourceTypes.- ####
    [InvokeApiTask](/docs/usercube/6.1/usercube/integration-guide/toolkit/xml-configuration/jobs/tasks/agent/invokeapitask/index.md)
    Tool to launch any Usercube API.- ####
    [InvokeAspectsTask](/docs/usercube/6.1/usercube/integration-guide/toolkit/xml-configuration/jobs/tasks/agent/invokeaspectstask/index.md)
    Call specific api in Usercube.- ####
    [InvokeExpressionTask](/docs/usercube/6.1/usercube/integration-guide/toolkit/xml-configuration/jobs/tasks/agent/invokeexpressiontask/index.md)
    Launches on agent side a powershell script given as input.- ####
    [InvokeSqlCommandTask](/docs/usercube/6.1/usercube/integration-guide/toolkit/xml-configuration/jobs/tasks/agent/invokesqlcommandtask/index.md)
    Takes as input an SQL file or an SQL command to output several CSV files that can be used by the
    collection.- ####
    [PrepareSynchronizationTask](/docs/usercube/6.1/usercube/integration-guide/toolkit/xml-configuration/jobs/tasks/agent/preparesynchronizationtask/index.md)
    Cleanses exported CSV files.
