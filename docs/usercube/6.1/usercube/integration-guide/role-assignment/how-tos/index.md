# How-Tos

    Use filters on dimensions and/or roles to restrict the assignment of a role or resource
    type.- ####
    [Infer Single Roles with a Composite Role](/docs/usercube/6.1/usercube/integration-guide/role-assignment/how-tos/infer-single-roles/index.md)
    Assign several single roles via the assignment of one composite role.- ####
    [Configure Indirect Permissions ](/docs/usercube/6.1/usercube/integration-guide/role-assignment/how-tos/configureindirectpermissions/index.md)
    Define indirect resource rules to use indirect permissions in Usercube.
