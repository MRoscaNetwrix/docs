# Reporting

The Reporting module is used to generate basic reports in CSV using
[SQuery](/docs/usercube/6.1/usercube/integration-guide/api/squery/index.md), or advanced
reports using the
[Business Intelligence module](/docs/usercube/6.1/usercube/integration-guide/toolkit/xml-configuration/business-intelligence/index.md).

[See more information about generating reports](/docs/usercube/6.1/usercube/user-guide/administrate/reporting/index.md).
