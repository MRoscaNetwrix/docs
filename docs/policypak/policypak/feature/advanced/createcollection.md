# Creating a Collection within the Wizard

As you work through the Feature Manager for Windows wizard, you will be asked to __Add policies to the existing collection__, or __Create a new collection__. If you create a new collection, you see the screen shown below. This process is the same as creating a collection manually, and the same options are available.
