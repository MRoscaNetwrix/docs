# XML Data Files from Endpoint Policy Manager Security Settings Manager

Endpoint Policy Manager Security Settings Manager will export the computer-side security within a GPO as an XML file. Right-click ```Computer Configuration | PolicyPak | Security Manager```, and select the only setting that is available in the menu, as shown in Figure 21. For full details on the Endpoint Policy Manager Security Settings Manager Export Wizard, see Book 10: [Security Settings Manager](/docs/policypak/policypak/securitysettings/overview.md).
