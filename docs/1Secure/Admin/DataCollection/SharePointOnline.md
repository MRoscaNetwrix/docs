# SharePoint Online Auditing

Netwrix 1Secure allows you to audit Office 365 organizations that have established modern authentication as their identity management approach, including support for [multi-factor authentication (MFA)](https://docs.microsoft.com/en-us/azure/active-directory/authentication/concept-mfa-howitworks "multi-factor authentication (MFA)"). See the Microsoft [App Registration and Configuration in Microsoft Entra ID](../../Configuration/EntraID/RegisterConfig.md "App Registration and Configuration in Microsoft Entra ID") article for additional information.
