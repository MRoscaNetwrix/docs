# SQL Server

## SQL Server Monitoring Scope

The product collects successful and failed logon attempts for Windows and SQL logons.

| Logon Type | Action |
| --- | --- |
| SQL logon | - Successful logon - Failed logon |
| Windows logon | - Successful logon - Failed logon |
