# Jobs

A job is defined via the `Job` tag to orchestrate tasks together, in order to perform specific
actions.

All [Tasks](/docs/identitymanager/6.2/integration-guide/toolkit/xml-configuration/jobs/tasks/index.md) types are child elements of jobs.

- [Tasks](/docs/identitymanager/6.2/integration-guide/toolkit/xml-configuration/jobs/tasks/index.md)
