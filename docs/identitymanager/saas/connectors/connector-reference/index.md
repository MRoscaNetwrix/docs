# References: Connectors

Connectors are the mechanisms that enable Identity Manager to read and write data to/from your
organization's systems. Here is a list of reference connectors:

- [Active Directory](/docs/identitymanager/saas/connectors/connector-reference/active-directory.md)

  Exports and fulfills users and groups from/to an Active Directory instance.

- [ Azure ](/docs/identitymanager/saas/connectors/connector-reference/azure.md)

  Exports Azure resources, role definitions and assignments.

- [ CSV ](/docs/identitymanager/saas/connectors/connector-reference/csv.md)

  Exports data from a CSV file.

- [ EasyVista ](/docs/identitymanager/saas/connectors/connector-reference/easyvista.md)

  Exports and fulfills users from/to an EasyVista-compliant system.

- [ EasyVista Ticket ](/docs/identitymanager/saas/connectors/connector-reference/easyvista-ticket.md)

  Opens tickets in EasyVista for manual provisioning.

- [ Google Workspace ](/docs/identitymanager/saas/connectors/connector-reference/google-workspace.md)

  Exports and fulfills users and groups from/to a Google Workspace instance.

- [ Home Folder ](/docs/identitymanager/saas/connectors/connector-reference/home-folder.md)

  Exports home folders' content.

- [InternalWorkflow](/docs/identitymanager/saas/connectors/connector-reference/internal-workflow.md)

  Triggers workflows in Identity Manager for a system's provisioning orders.

- [Internal Resources](/docs/identitymanager/saas/connectors/connector-reference/internal-resources.md)

  Opens manual provisioning tickets in Identity Manager.

- [JSON](/docs/identitymanager/saas/connectors/connector-reference/json.md)

  Generates JSON files for each provisioning order.

- [ LDAP ](/docs/identitymanager/saas/connectors/connector-reference/ldap.md)

  Exports and fulfills entries from/to a LDAP-compliant system.

- [LDIF](/docs/identitymanager/saas/connectors/connector-reference/ldif.md)

  Exports entries from a LDIF file.

- [ Microsoft Entra ID](/docs/identitymanager/saas/connectors/connector-reference/microsoft-entra-id.md)

  Exports and fulfills user and groups from/to a Microsoft Entra ID instance.

- [ Microsoft Excel ](/docs/identitymanager/saas/connectors/connector-reference/excel.md)

  Exports datasheets from a Microsoft Excel (XLSX) file.

- [ Microsoft Exchange ](/docs/identitymanager/saas/connectors/connector-reference/microsoft-exchange.md)

  Exports mailboxes from a Microsoft Exchange instance.

- [ OData ](/docs/identitymanager/saas/connectors/connector-reference/odata.md)

  Exports and fulfills entries from/to an OData instance.

- [Okta](/docs/identitymanager/saas/connectors/connector-reference/okta.md)

  Exports and fulfills entries from/to an Okta instance.

- [ OpenLDAP ](/docs/identitymanager/saas/connectors/connector-reference/openldap.md)

  Exports and fulfills entries from/to an OpenLDAP directory.

- [ PowerShellProv ](/docs/identitymanager/saas/connectors/connector-reference/powershell-prov.md)

  Writes to an external system via a PowerShell script.

- [ PowerShellSync ](/docs/identitymanager/saas/connectors/connector-reference/powershell-sync.md)

  Exports data from an external system via a Powershell script.

- [ RACF ](/docs/identitymanager/saas/connectors/connector-reference/racf.md)

  Exports users and profiles from a RACF file.

- [ Robot Framework ](/docs/identitymanager/saas/connectors/connector-reference/robot-framework.md)

  Writes to an external system via a Robot Framework script.

- [SAP ERP 6.0 and SAP S4/HANA](/docs/identitymanager/saas/connectors/connector-reference/sap-erp6.md)

  Exports and fulfills users and roles from/to a SAP ERP 6.0 or SAP S4/HANA instance.

- [ SAP Netweaver ](/docs/identitymanager/saas/connectors/connector-reference/sap-netweaver.md)

  Exports and fulfills users and roles from/to a SAP Netweaver instance.

- [SCIM](/docs/identitymanager/saas/connectors/connector-reference/scim.md)

  Exports and fulfills entities from/to a SCIM-compliant application.

- [ ServiceNow ](/docs/identitymanager/saas/connectors/connector-reference/servicenow-entity.md)

  Exports and fulfills any data from/to a ServiceNow CMDB.

- [ ServiceNowTicket ](/docs/identitymanager/saas/connectors/connector-reference/servicenow-ticket.md)

  Opens tickets in ServiceNow for manual provisioning.

- [ SharedFolders ](/docs/identitymanager/saas/connectors/connector-reference/shared-folder.md)

  Exports users and permissions from Windows shared folders.

- [SharePoint](/docs/identitymanager/saas/connectors/connector-reference/sharepoint.md)

  Exports sites, folders, groups and permissions from a SharePoint instance.

- [ Sql ](/docs/identitymanager/saas/connectors/connector-reference/sql.md)

  Exports data from one of various Database Management Systems.

- [ Sql Server Entitlements ](/docs/identitymanager/saas/connectors/connector-reference/sql-server-entitlements.md)

  Exports entitlements from Microsoft SQL Server.

- [ Top Secret ](/docs/identitymanager/saas/connectors/connector-reference/top-secret.md)

  Exports users and profiles from a Top Secret (TSS) instance.

- [ Workday ](/docs/identitymanager/saas/connectors/connector-reference/workday.md)

  Exports users and groups from a Workday instance.
