# Action Queue

The Action Queue Overview shows any pending or in-progress actions taken by the Threat Manager Action Service.

![actionqueue](../../../../../../static/img/product_docs/threatmanager/threatmanager/administration/configuration/systemhealth/actionqueue.webp)

This includes Honeytoken deployments and Threat Response Playbook executions. Any actions in the action queue may be stopped by clicking the __Stop__ button.
