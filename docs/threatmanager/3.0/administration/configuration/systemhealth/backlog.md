# Backlog

The Backlog overview displays a summary of all threats and system jobs with the events in queue to
be processed. It also displays other job information depending on the job type.

![Backlog Overview](/img/product_docs/threatmanager/3.0/administration/configuration/systemhealth/backlogoverview.webp)

In Threat Manager, jobs are used for threat evaluation, maintenance tasks, and operational
procedures such as email and SIEM notifications.
