# Setup

## Requirements

Netwrix Password Secure Mobile Apps automatically synchronize with an existing Netwrix Password
Secure database. The
[Web Application](/versioned_docs/passwordsecure_9.1/passwordsecure/configuration/web_applicaiton/web_application.md)
is used as the interface for this. This must therefore be installed. In addition, the database must
be enabled for use with mobile devices on the
[Server Manager](/versioned_docs/passwordsecure_9.1/passwordsecure/configuration/server_manager/server_manger.md).
