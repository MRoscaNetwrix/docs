# Database settings

To open the settings of a database, select it and click on "Settings" in the ribbon. Alternatively
you can open the context menu with the right mouse button and click on "Properties". In the next
step you will be asked to enter your admin password. After that a window with the settings will
open.

#### Settings

You can now make the following settings:

- Authentication
- [Multifactor Authentication](/docs/passwordsecure/9.1/configuration/authentication/multifactor-authentication/multifactor-overview.md)
- [Session timeout     ](/docs/passwordsecure/9.1/configuration/server-manager/session-timeout.md)
- [HSM connection via PKCS # 11](/docs/passwordsecure/9.1/configuration/server-manager/hsm-connection.md)
- Automatic cleanup
- SAML configuration
- Deletion of users
- More options
