# Mobile Apps

#### Required Version

**CAUTION:** Our mobile apps are only supported on devices with the official OS (no jailbreak, not
rooted).

|                  |         |             |
| ---------------- | ------- | ----------- |
| OS               | Minimum | Recommended |
| iOS (Apple)      | 17.7.1  | 18.1        |
| iPadOS (Apple)   | 17.7.1  | 18.1        |
| Android (Google) | 13      | 15          |
