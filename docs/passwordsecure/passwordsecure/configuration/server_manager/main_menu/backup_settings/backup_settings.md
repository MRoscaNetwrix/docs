# Backup settings

## What are backup settings?

Within the backup settings the default values for the execution of backups can be defined.

![Backup settings](../../../../../../../static/img/product_docs/passwordsecure/passwordsecure/configuration/server_manager/main_menu/backup_settings/installation_with_parameters_255-en.webp)

#### Interval settings

The interval for backups can be customized as needed. A separate assistant is available for this
purpose.

![define interval in backup settings](../../../../../../../static/img/product_docs/passwordsecure/passwordsecure/configuration/server_manager/main_menu/backup_settings/installation_with_parameters_256-en.webp)
