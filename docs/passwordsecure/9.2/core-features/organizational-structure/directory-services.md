# Directory services

It is possible to use existing user and group structures from external directories with Netwrix
Password Secure.

Choose your preferred integration method:

- [Microsoft Entra ID connection](/docs/passwordsecure/9.2/core-features/organizational-structure/entra-id-integration.md)

- [Active Directory link](/docs/passwordsecure/9.2/core-features/organizational-structure/active-directory-integration.md)
