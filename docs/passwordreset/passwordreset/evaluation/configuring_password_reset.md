# Configuring Password Reset

In the previous section, you used Password Reset with a default configuration. You can use the Configuration Console to edit the configuration settings. Click Start > Netwrix Password Reset > NPR Configuration Console to open the console.

![configuring_npr_1](/img/product_docs/passwordpolicyenforcer/passwordreset/administration/configuring_npr_1.png)

The Configuration Console has a tabbed layout. Click the tabs along the top to see the various settings. Most of the settings are self-explanatory. Press __F1__ on any of the tabs to see the help page for the current tab.

Configuration changes are applied when you click __Apply__ or __OK__. Clicking __OK__ also closes the Configuration Console.

__NOTE:__ Password Reset includes a 30-day evaluation license for up to 50 users. Please[ contact Netwrix support](mailto:support@netwrix.com) if you would like to evaluate Netwrix Password Reset with more than 50 users.
