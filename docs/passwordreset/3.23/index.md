# Password Reset 3.23

> Secure self-service password management for enterprise productivity

Password Reset 3.23 empowers users to securely reset their own passwords and unlock accounts without IT assistance. This self-service solution reduces help desk workload while maintaining enterprise-grade security through multi-factor authentication and intelligent verification methods, ensuring legitimate users can quickly regain access to their accounts while keeping unauthorized users out.

## Key Features

- **Multi-Factor Authentication**: Support for SMS, email, security questions, and biometric verification methods
- **Account Unlock Capabilities**: Enable users to unlock their accounts without resetting passwords
- **Mobile-First Design**: Responsive web interface optimized for smartphones and tablets
- **Integration Flexibility**: Seamless integration with Active Directory, Azure AD, and LDAP directories

## Benefits

- **Reduced IT Costs**: Eliminate up to 40% of help desk calls related to password resets
- **Improved Productivity**: Users regain access immediately without waiting for IT support
- **Enhanced Security**: Multi-factor authentication prevents unauthorized password resets
- **24/7 Availability**: Users can reset passwords anytime, from anywhere

## What's New in Version 3.23

- Enhanced mobile app with biometric authentication support
- Improved Azure AD B2C integration capabilities
- New adaptive authentication based on risk scoring
- Expanded language support for global deployments
