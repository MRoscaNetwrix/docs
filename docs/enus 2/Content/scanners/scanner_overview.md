# Scanner Overview

The Scanner menu is located on the Netwrix Dashboard page.

* [Scheduler](scheduler "Open the Scheduler topic") schedules automatic scans.
* [Manual Scanners](../installing_strongpoint/running_scanner "Manual Scanners") accesses the list of scanners where you can select one or more to run. **Manual Scanners** can also be run from **Netwrix Dashboard**.
* [Daily Scan Configuration](daily_scan "Daily Scan Configuration") enables an administrator to select object types for daily scanning, instead of defaulting to all types.
* [Field Level Security Scanner](field_level_scanner "Open the Field-Level Scanner topic") displays all fields using the selected Salesforce object.
