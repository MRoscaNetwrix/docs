# User Lifecycle Commands

This section covers the cmdlets for performing user lifecycle tasks such as:

- [Extend-User](/versioned_docs/groupid_11.0/groupid/managementshell/userlifecycle/extenduser.md):
  activates a user account for a specific number of days
- [Get-Status](/versioned_docs/groupid_11.0/groupid/managementshell/userlifecycle/getstatus.md):
  provides the status of a user as per the profile validation criteria.
- [Reinstate-User](/versioned_docs/groupid_11.0/groupid/managementshell/userlifecycle/reinstateuser.md):
  activates or disables a user.
- [Terminate-DirectReports](/versioned_docs/groupid_11.0/groupid/managementshell/userlifecycle/terminatedirectreports.md):
  terminates direct reports of a user.
- [Transfer-DirectReports ](/versioned_docs/groupid_11.0/groupid/managementshell/userlifecycle/transferdirectreports.md):
  transfers direct reports of a user.
