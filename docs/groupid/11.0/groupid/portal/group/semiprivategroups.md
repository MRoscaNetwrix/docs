# Semi Private Groups

This view lists only the semi-private groups created using GroupID in the connected identity store.
It does not list expired or deleted semi private groups. To view the expired or deleted groups,
select the [Expired Groups](/docs/groupid/11.0/groupid/portal/group/allexpiredgroups.md)
or [Deleted Groups](/docs/groupid/11.0/groupid/portal/group/recyclebin/overview.md)
respectively.

You can
[Modify Search Directory](/docs/groupid/11.0/groupid/portal/group/allgroups.md#modify-search-directory)
to search semi private groups and add
[Filter All Groups](/docs/groupid/11.0/groupid/portal/group/allgroups.md#filter-all-groups)
by clicking **Private Group Grid Filters**. All the semi private groups matching the filters will be
displayed.
