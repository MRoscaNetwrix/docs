# Group Type page

Select the type of group you want to create and click **Next**. Options are:

- [Create a Static Group](/docs/groupid/11.0/groupid/portal/group/create/activedirectory/group.md#create-a-static-group)
- [Create a Smart Group](/docs/groupid/11.0/groupid/portal/group/create/activedirectory/group.md#create-a-smart-group)
- [Create a Password Expiry Group](/docs/groupid/11.0/groupid/portal/group/create/activedirectory/group.md#create-a-password-expiry-group)
  (not supported in Microsoft Entra ID)
- [Create a Dynasty using the Organizational/Geographical/Custom template](/docs/groupid/11.0/groupid/portal/group/dynasty/activedirectory/createdynasty.md#create-a-dynasty-using-the-organizationalgeographicalcustom-template)
- [Create a Dynasty using the Managerial template](/docs/groupid/11.0/groupid/portal/group/dynasty/activedirectory/createdynasty.md#create-a-dynasty-using-the-managerial-template)
- [Create Teams](/docs/groupid/11.0/groupid/portal/group/teams/create.md) (for Microsoft
  Entra ID only)
