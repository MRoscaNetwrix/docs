# Smart Groups

This view lists only the Smart Groups created using Directory Manager in the connected identity store. It does not list expired or deleted Smart Groups. To view the expired or deleted groups, select the [Expired Groups](/docs/groupid/groupid/portal/group/allexpiredgroups.md) or
[Deleted Groups](/docs/groupid/groupid/portal/group/recyclebin/overview.md) respectively.

You can [Modify Search Directory](/docs/groupid/groupid/portal/group/allgroups.md#modify-search-directory) to search smart groups and add [Filter All Groups](/docs/groupid/groupid/portal/group/allgroups.md#filter-all-groups)by clicking __Smart Group Grid Filter__.
All the smart groups matching the filters will be displayed.
