# Group Type page

Select the type of group you want to create and click __Next__. Options are:

- [Create a Static Group](/docs/product_docs/groupid/groupid/portal/group/create/activedirectory/group.md#Create-a-Static-Group)
- [Create a Smart Group](/docs/product_docs/groupid/groupid/portal/group/create/activedirectory/group.md#Create-a-Smart-Group)
- [Create a Password Expiry Group](/docs/product_docs/groupid/groupid/portal/group/create/activedirectory/group.md#Create-a-Password-Expiry-Group) (not supported in Microsoft Entra ID)
- [Create a Dynasty using the Organizational/Geographical/Custom template](/docs/product_docs/groupid/groupid/portal/group/dynasty/activedirectory/createdynasty.md#Create-a-Dynasty-using-the-OrganizationalGeographicalCustom-template)
- [Create a Dynasty using the Managerial template](/docs/product_docs/groupid/groupid/portal/group/dynasty/activedirectory/createdynasty.md#Create-a-Dynasty-using-the-Managerial-template)
- [Create Teams](/docs/product_docs/groupid/groupid/portal/group/teams/create.md) (for Microsoft Entra ID only)
