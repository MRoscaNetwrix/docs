# Teams

This page lists all the Microsoft Teams groups.

You can [Modify Search Directory](/docs/groupid/groupid/portal/group/allgroups.md#Modify-Search-Directory) to search teams and add [Filter All Groups](/docs/groupid/groupid/portal/group/allgroups.md#Filter-All-Groups)by clicking __Teams Grid Filters__.
All the teams matching the filters will be displayed.
