# My Smart Groups

This page lists only the Smart Groups that you are primary owner of. To include the groups for which you are an additional owner, select the __Display additional group ownership in My Smart Groups__ check box on the [Portal Settings](/docs/product_docs/groupid/groupid/portal/setting/portal.md) panel.

You can [Modify Search Directory](/docs/product_docs/groupid/groupid/portal/group/allgroups.md#Modify-Search-Directory) to search your smart groups and add [Filter All Groups](/docs/product_docs/groupid/groupid/portal/group/allgroups.md#Filter-All-Groups)by clicking __Smart Group Grid Filters__.
All the smart groups matching the filters will be displayed.
