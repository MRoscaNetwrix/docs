# Private Groups

This view lists only the private groups created using Directory Manager in the connected identity store. It does not list expired or deleted private groups. To view the expired or deleted groups, select the [Expired Groups](/docs/groupid/groupid/portal/group/allexpiredgroups.md) or
[Deleted Groups](/docs/groupid/groupid/portal/group/recyclebin/overview.md) respectively.

You can [Modify Search Directory](/docs/groupid/groupid/portal/group/allgroups.md#modify-search-directory) to search private groups and add [Filter All Groups](/docs/groupid/groupid/portal/group/allgroups.md#filter-all-groups) by clicking __Private Group Grid Filters__.
All the private groups matching the filters will be displayed.
