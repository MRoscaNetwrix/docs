# My Dynasties

This page lists the Dynasties you are the primary owner of. To include the groups for which you are an additional owner, select the __Display additional group ownership in My Dynasties__ check box on the [Portal Settings](/docs/product_docs/groupid/groupid/portal/setting/portal.md) panel.

You can [Modify Search Directory](/docs/product_docs/groupid/groupid/portal/group/allgroups.md#Modify-Search-Directory) to search dynasties and add [Filter All Groups](/docs/product_docs/groupid/groupid/portal/group/allgroups.md#Filter-All-Groups)by clicking __My Dynasties Grid Filters__.
All the dynasties matching the filters will be displayed.
