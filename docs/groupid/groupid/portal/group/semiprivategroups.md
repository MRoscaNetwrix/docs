# Semi Private Groups

This view lists only the semi-private groups created using Directory Manager in the connected identity store. It does not list expired or deleted semi private groups. To view the expired or deleted groups, select the [Expired Groups](/docs/groupid/groupid/portal/group/allexpiredgroups.md) or
[Deleted Groups](/docs/groupid/groupid/portal/group/recyclebin/overview.md) respectively.

You can [Modify Search Directory](/docs/groupid/groupid/portal/group/allgroups.md#Modify-Search-Directory) to search semi private groups and add [Filter All Groups](/docs/groupid/groupid/portal/group/allgroups.md#Filter-All-Groups) by clicking __Private Group Grid Filters__.
All the semi private groups matching the filters will be displayed.
