# My Teams

This page lists all the Microsoft Teams groups that you own.

You can [Modify Search Directory](/docs/product_docs/groupid/groupid/portal/group/allgroups.md#Modify-Search-Directory) to search teams and add [Filter All Groups](/docs/product_docs/groupid/groupid/portal/group/allgroups.md#Filter-All-Groups)by clicking __My Teams Grid Filters__.
All the teams matching the filters will be displayed.
