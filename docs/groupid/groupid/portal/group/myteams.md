# My Teams

This page lists all the Microsoft Teams groups that you own.

You can [Modify Search Directory](/docs/groupid/groupid/portal/group/allgroups.md#modify-search-directory) to search teams and add [Filter All Groups](/docs/groupid/groupid/portal/group/allgroups.md#filter-all-groups)by clicking __My Teams Grid Filters__.
All the teams matching the filters will be displayed.
