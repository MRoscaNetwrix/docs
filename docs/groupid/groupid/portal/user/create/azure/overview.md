# Create Microsoft Entra ID User Objects

In a Microsoft Entra ID identity stores, you can create the following types of users:

- User – See the [Create a Microsoft Entra ID User](/docs/product_docs/groupid/groupid/portal/user/create/azure/user.md) topic for additional information on how to create a user in an Microsoft Entra ID identity store.
- Mailbox – See the [Create a Microsoft Entra ID Mailbox](/docs/product_docs/groupid/groupid/portal/user/create/azure/mailbox/mailbox.md) topic for additional information on how to create a mailbox in an Microsoft Entra ID identity store.
