# Contact APIs

Directory Manager provides the following APIs to perform contact-specific functions:

- [Create a Contact](/docs/product_docs/groupid/groupid/api/contact/createcontact.md)
- [Delete a Contact](/docs/product_docs/groupid/groupid/api/contact/deletecontact.md)
- [Delete Contacts](/docs/product_docs/groupid/groupid/api/contact/deletecontacts.md)
- [Get a Contact](/docs/product_docs/groupid/groupid/api/contact/getcontact.md)
- [Get Contacts](/docs/product_docs/groupid/groupid/api/contact/getcontacts.md)
- [Update a Contact](/docs/product_docs/groupid/groupid/api/contact/updatecontact.md)

NOTE: The contact object type is not supported in a Microsoft Entra ID based identity store.
