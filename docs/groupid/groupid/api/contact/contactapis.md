# Contact APIs

Directory Manager provides the following APIs to perform contact-specific functions:

- [Create a Contact](createcontact.md)
- [Delete a Contact](deletecontact.md)
- [Delete Contacts](deletecontacts.md)
- [Get a Contact](getcontact.md)
- [Get Contacts](getcontacts.md)
- [Update a Contact](updatecontact.md)

NOTE: The contact object type is not supported in a Microsoft Entra ID based identity store.
