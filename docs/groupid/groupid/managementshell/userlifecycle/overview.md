# User Lifecycle Commands

This section covers the cmdlets for performing user lifecycle tasks such as:

- [Extend-User](extenduser.md) – Activates a user account for a specific number of days
- [Get-Status](getstatus.md) – Provides the status of a user as per the profile validation criteria.
- [Reinstate-User](reinstateuser.md) – Activates or disables a user.
- [Terminate-DirectReports](terminatedirectreports.md) – Terminates direct reports of a user.
- [Transfer-DirectReports ](transferdirectreports.md) – Transfers direct reports of a user.
