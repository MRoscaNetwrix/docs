# General Commands

You can use the following Management Shell cmdlets to perform tasks such as:

- [Get-Computer](/docs/groupid/groupid/managementshell/general/getcomputer.md) – Provides information about a computer object.
- [Get-ConnectedStoreInformation](/docs/groupid/groupid/managementshell/general/getconnectedstoreinformation.md) – Provides information about the connected identity store.
- [Get-ConnectedUser](/docs/groupid/groupid/managementshell/general/getconnecteduser.md) – Provides information about the connected user.
- [Get-GroupIdInformation](/docs/groupid/groupid/managementshell/general/getgroupidinformation.md) – Provides information about GroupID.
- [Get-ImanamiCommand](/docs/groupid/groupid/managementshell/general/getimanamicommand.md) – Provides basic information about GroupID Management Shell cmdlets.
- [Get-ReplicationStatus](/docs/groupid/groupid/managementshell/general/getreplicationstatus.md) – Provides the replication status of objects in an identity store.
- [Get-TombStoneObject](/docs/groupid/groupid/managementshell/general/gettombstoneobject.md) – Displays information about tombstone objects.
- [Invoke-Replication](/docs/groupid/groupid/managementshell/general/invokereplication.md) – Starts the replication process for all the identity stores or for a specific identity store.
- [New-Container](/docs/groupid/groupid/managementshell/general/newcontainer.md) – Creates a new organizational unit.
- [Remove-Container](/docs/groupid/groupid/managementshell/general/removecontainer.md) – Removes an empty organizational unit.
- [Restore-TombStoneObject](/docs/groupid/groupid/managementshell/general/restoretombstoneobject.md) – Restores tombstone objects from the directory.
- [Send-Notification](/docs/groupid/groupid/managementshell/general/sendnotification.md) – Sends notifications to a group or a user.
