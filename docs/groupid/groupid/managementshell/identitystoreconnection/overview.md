# Identity Store Connection Commands

This section covers cmdlets for establishing a connection with an identity store.

- [Connect-IdentityStore](/docs/product_docs/groupid/groupid/managementshell/identitystoreconnection/connectidentitystore.md) – Connects to an identity store using the authentication mode mentioned.
- [Get-Token](/docs/product_docs/groupid/groupid/managementshell/identitystoreconnection/gettoken.md) – Gets a token from the Directory Manager Security service.
