# Contact Commands

Directory Manager provides the following cmdlets to perform contact-related tasks, such as:

- [Get-Contact](getcontact.md) – Retrieves a contact that matches the given criteria.
- [New-Contact](newcontact.md) – Creates a new contact.
- [Remove-Contact](removecontact.md) – Removes a contact from the directory.
- [Set-Contact](setcontact.md) – Modifies a contact in the directory.
