# Dynasty Commands

This section covers the following cmdlets for managing Dynasties.

- [New-Dynasty](/docs/groupid/11.1/groupid/managementshell/dynasty/newdynasty.md) – Creates a new Dynasty.
- [Set-Dynasty](/docs/groupid/11.1/groupid/managementshell/dynasty/setdynasty.md) – Modifies a Dynasty or its children.
