# Mailbox Commands

This section covers cmdlets for performing mailbox-specific tasks such as:

- [Get-Mailbox](/docs/groupid/11.1/groupid/managementshell/mailbox/getmailbox.md) – Retrieves a mailbox.
- [New-Mailbox](/docs/groupid/11.1/groupid/managementshell/mailbox/newmailbox.md) – Creates a new mailbox.
- [Remove-Mailbox](/docs/groupid/11.1/groupid/managementshell/mailbox/removemailbox.md) – Deletes a mailbox.
- [Set-Mailbox](/docs/groupid/11.1/groupid/managementshell/mailbox/setmailbox.md) – Modifies a mailbox.
