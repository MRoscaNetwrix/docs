# Mailbox Commands

This section covers cmdlets for performing mailbox-specific tasks such as:

- [Get-Mailbox](getmailbox.md) – Retrieves a mailbox.
- [New-Mailbox](newmailbox.md) – Creates a new mailbox.
- [Remove-Mailbox](removemailbox.md) – Deletes a mailbox.
- [Set-Mailbox](setmailbox.md) – Modifies a mailbox.
