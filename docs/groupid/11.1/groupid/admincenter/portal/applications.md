# Directory Manage Applications

Using Admin Center, you can create and manage the following Directory Manager applications:

- [ Directory Manager Portal](overview.md)
- [Data Service](../service/dataservice/overview.md)
- [Security Service](../service/securityservice/overview.md)

Moreover, you can manage some basic deployment and log settings for the following applications:

- [Admin Center](../service/admincenter.md)
- [Replication Service](../service/replicationservice.md)
- [Email Service](../service/emailservice.md)
- [Scheduler Service](../service/schedulerservice.md)

See the [Services](../service/overview.md) topic for additional information.
