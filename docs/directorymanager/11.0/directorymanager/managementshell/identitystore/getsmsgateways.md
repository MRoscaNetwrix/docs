# Get-SmsGateways

The commandlet **Get-SmsGateways** provides information of the SMS gateways configured in GroupID.

## Syntax

```
Get-SmsGateways
[<CommonParameters>]
```

## Required Parameter

- None

Example:

This example lists all the configured SMS gateways in GroupID.

```
Get-SmsGateways
```
