# User Lifecycle Commands

This section covers the cmdlets for performing user lifecycle tasks such as:

- [Extend-User](/docs/directorymanager/11.0/directorymanager/managementshell/userlifecycle/extenduser.md):
  activates a user account for a specific number of days
- [Get-Status](/docs/directorymanager/11.0/directorymanager/managementshell/userlifecycle/getstatus.md):
  provides the status of a user as per the profile validation criteria.
- [Reinstate-User](/docs/directorymanager/11.0/directorymanager/managementshell/userlifecycle/reinstateuser.md):
  activates or disables a user.
- [Terminate-DirectReports](/docs/directorymanager/11.0/directorymanager/managementshell/userlifecycle/terminatedirectreports.md):
  terminates direct reports of a user.
- [Transfer-DirectReports ](/docs/directorymanager/11.0/directorymanager/managementshell/userlifecycle/transferdirectreports.md):
  transfers direct reports of a user.
