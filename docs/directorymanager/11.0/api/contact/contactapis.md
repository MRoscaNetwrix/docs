# Contact APIs

GroupID provides the following APIs to perform contact-specific functions:

- [Create a Contact](/docs/directorymanager/11.0/api/contact/createcontact.md)
- [Delete a Contact](/docs/directorymanager/11.0/api/contact/deletecontact.md)
- [Delete Contacts](/docs/directorymanager/11.0/api/contact/deletecontacts.md)
- [Get a Contact](/docs/directorymanager/11.0/api/contact/getcontact.md)
- [Get Contacts](/docs/directorymanager/11.0/api/contact/getcontacts.md)
- [Update a Contact](/docs/directorymanager/11.0/api/contact/updatecontact.md)

NOTE: The contact object type is not supported in a Microsoft Entra ID based identity store.
