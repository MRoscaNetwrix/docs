# My Smart Groups

This page lists only the Smart Groups that you are primary owner of. To include the groups for which
you are an additional owner, select the **Display additional group ownership in My Smart Groups**
check box on the [Portal Settings](/docs/directorymanager/11.0/welcome/generalfeatures/portal.md)
panel.

You can
[Modify Search Directory](/docs/directorymanager/11.0/welcome/group/allgroups/allgroups.md#modify-search-directory)
to search your smart groups and add
[Filter All Groups](/docs/directorymanager/11.0/welcome/group/allgroups/allgroups.md#filter-all-groups)by
clicking **Smart Group Grid Filters**. All the smart groups matching the filters will be displayed.
