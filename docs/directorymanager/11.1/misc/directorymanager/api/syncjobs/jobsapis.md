# Synchronize Jobs APIs

Directory Manager Synchronize is used for transferring data from one data source to another. The
data sources may include directory servers, databases or files. The following APIs can be used for
performing Synchronize jobs-specific functions:

- [Create a New Job](/docs/directorymanager/11.1/misc/directorymanager/api/syncjobs/createjob.md)
- [Create a New Job Collection](/docs/directorymanager/11.1/misc/directorymanager/api/syncjobs/createnewjobcollection.md)
- [Delete Jobs](/docs/directorymanager/11.1/misc/directorymanager/api/syncjobs/deletejob.md)
- [Delete Job Collections](/docs/directorymanager/11.1/misc/directorymanager/api/syncjobs/deletejobcollections.md)
- [Get Job Collections Details](/docs/directorymanager/11.1/misc/directorymanager/api/syncjobs/getcollectionsdetails.md)
- [Get Job Collection Details By Job Collection ID](/docs/directorymanager/11.1/misc/directorymanager/api/syncjobs/getjcdetailsbyjcid.md)
- [Get Jobs](/docs/directorymanager/11.1/misc/directorymanager/api/syncjobs/getjobs.md)
- [Get Job Collections](/docs/directorymanager/11.1/misc/directorymanager/api/syncjobs/getjobcollections.md)
- [Get a Job Details](/docs/directorymanager/11.1/misc/directorymanager/api/syncjobs/getjobsdetails.md)
- [Get Jobs Names ](/docs/directorymanager/11.1/misc/directorymanager/api/syncjobs/getjobsname.md)
- [Update a Job Collection](/docs/directorymanager/11.1/misc/directorymanager/api/syncjobs/updatjobcollection.md)
