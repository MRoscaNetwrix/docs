# Get-SmsGateways

The commandlet Get-SmsGateways provides information of the SMS gateways configured in Directory
Manager.

## Syntax

```
Get-SmsGateways
[<CommonParameters>]
```

## Required Parameter

- None

Example:

This example lists all the configured SMS gateways in Directory Manager.

```
Get-SmsGateways
```
