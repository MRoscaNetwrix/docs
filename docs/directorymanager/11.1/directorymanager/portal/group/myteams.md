# My Teams

This page lists all the Microsoft Teams groups that you own.

You can [Modify Search Directory](/docs/directorymanager/11.1/directorymanager/portal/group/allgroups.md#modify-search-directory) to search teams and add
[Filter All Groups](/docs/directorymanager/11.1/directorymanager/portal/group/allgroups.md#filter-all-groups)by clicking **My Teams Grid Filters**. All the
teams matching the filters will be displayed.
