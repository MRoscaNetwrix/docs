# Identity Store Commands

This section covers cmdlets for performing identity store-related tasks such as:

- [Clear-MessagingServer](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/clearmessagingserver.md) – Removes a configured messaging server.
- [Clear-Notifications](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/clearnotifications.md) – Removes notification settings of an identity store.
- [Clear-SmtpServer](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/clearsmtpserver.md) – Removes a configured SMTP server of an identity store.
- [Get-AvailableMessagingServers](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/getavailablemessagingservers.md) – Retrieves messaging servers for
  the configured messaging provider.
- [Get-Client](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/getclient.md) – Lists information about the Directory Manager clients.
- [Get-IdentityStore](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/getidentitystore.md) – Retrieves information about an identity store.
- [Get-IdentityStoreRoles](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/getidentitystoreroles.md) – Retrieves information about the security
  roles in an identity store
- [Get-LogSettings](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/getlogsettings.md) – Provides information about the global log settings of an
  identity store.
- [Get-RolePermissionNames](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/getrolepermissionnames.md) – Lists the permissions assigned to the
  logged-in user.
- [Get-SchemaAttributes](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/getschemaattributes.md) – Lists schema attributes available for an identity
  store.
- [Get-SmsGateways](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/getsmsgateways.md) – Provides information of the configured SMS gateways.
- [Get-UserRole](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/getuserrole.md) – Displays role information of a user in an identity store.
- [New-IdentityStore](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/newidentitystore.md) – Creates a new identity store.
- [Remove-IdentityStore](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/removeidentitystore.md) – Removes an identity store from Directory Manager.
- [Send-TestNotification](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/sendtestnotification.md) – Sends a test notification.
- [Set-IdentityStore](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/setidentitystore.md) – Modifies an identity store configuration.
- [Set-IdentityStoreRole](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/setidentitystorerole.md) – Modifies properties of a security role in an
  identity store.
- [Set-MessagingServer](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/setmessagingserver.md) – Configures a messaging server in an identity store.
- [Set-Notifications](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/setnotifications.md) – Modifies notification settings of an identity store.
- [Set-SmtpServer](/docs/directorymanager/11.1/directorymanager/managementshell/identitystore/setsmtpserver.md) – Configures an SMTP server of an identity store.
