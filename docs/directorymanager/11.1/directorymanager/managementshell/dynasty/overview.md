# Dynasty Commands

This section covers the following cmdlets for managing Dynasties.

- [New-Dynasty](/docs/directorymanager/11.1/directorymanager/managementshell/dynasty/newdynasty.md) – Creates a new Dynasty.
- [Set-Dynasty](/docs/directorymanager/11.1/directorymanager/managementshell/dynasty/setdynasty.md) – Modifies a Dynasty or its children.
