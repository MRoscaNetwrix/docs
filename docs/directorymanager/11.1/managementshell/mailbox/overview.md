# Mailbox Commands

This section covers cmdlets for performing mailbox-specific tasks such as:

- [Get-Mailbox](/docs/directorymanager/11.1/managementshell/mailbox/getmailbox.md) – Retrieves a mailbox.
- [New-Mailbox](/docs/directorymanager/11.1/managementshell/mailbox/newmailbox.md) – Creates a new mailbox.
- [Remove-Mailbox](/docs/directorymanager/11.1/managementshell/mailbox/removemailbox.md) – Deletes a mailbox.
- [Set-Mailbox](/docs/directorymanager/11.1/managementshell/mailbox/setmailbox.md) – Modifies a mailbox.
