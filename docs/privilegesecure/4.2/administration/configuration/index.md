# Configuration Interface

The Configuration interface provides information and management options for advanced configuration
settings.

![Configuration Interface](/img/product_docs/threatprevention/threatprevention/reportingmodule/interface.webp)

Expand the Configuration menu in the Navigation pane for related pages:

- [Service Accounts Page](/docs/privilegesecure/4.2/administration/configuration/services/service-accounts.md) — Add or modify service accounts
- Service Nodes:

  - [Service Nodes Page](/docs/privilegesecure/4.2/administration/configuration/services/service-nodes.md) — View the status and details of Privilege Secure
    Services
  - [Scheduled Tasks Page](/docs/privilegesecure/4.2/administration/configuration/system-settings/scheduled-tasks.md) — View or modify recurring tasks

- System Settings — Modify the system settings:

  - [Action Service Settings Page](/docs/privilegesecure/4.2/administration/configuration/services/action-service-settings.md)
  - [Database Page](/docs/privilegesecure/4.2/administration/configuration/system-settings/database-settings.md)
  - [Email Configuration Page](/docs/privilegesecure/4.2/administration/configuration/system-settings/email-configuration.md)
  - [Global Settings Page](/docs/privilegesecure/4.2/administration/configuration/system-settings/global-settings.md)
  - [Local Account Password Options Page](/docs/privilegesecure/4.2/administration/credentials/password-policies/local-account-options.md)
  - [Password History Options Page](/docs/privilegesecure/4.2/administration/credentials/password-policies/password-history-options.md)
  - [Local Account Password Options Page](/docs/privilegesecure/4.2/administration/credentials/password-policies/local-account-options.md)
  - [Services Page](/docs/privilegesecure/4.2/administration/configuration/services/index.md)

- [Authentication Page](/docs/privilegesecure/4.2/administration/configuration/authentication/authentication-settings.md) — Add or modify multi-factor authentication (MFA)
- [Integration Connectors Page](/docs/privilegesecure/4.2/administration/configuration/integrations/index.md) — Configure settings for integration
  with other applications
- SIEM:

  - [SIEM Server Page](/docs/privilegesecure/4.2/administration/configuration/system-settings/siem-configuration/siem-server.md) — Add or modify SIEM servers
  - [SIEM Templates Page](/docs/privilegesecure/4.2/administration/configuration/system-settings/siem-configuration/siem-templates.md) — Add or modify SIEM templates
