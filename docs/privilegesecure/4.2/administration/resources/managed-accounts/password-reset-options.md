# Password Reset Options Window

The Password Reset option control password resets for the selected account.

Click the **Password Reset Options** button to open the window and configure the following:

![passwordresetoptions](/img/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/policy/window/resources/passwordresetoptions.webp)

Select options to rotate the account password when the session starts or ends.

When clicking **Okay** button, the password reset policy is applied to an account.
