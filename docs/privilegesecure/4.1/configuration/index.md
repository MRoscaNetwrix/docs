# Configuration Interface

The Configuration interface provides information and management options for advanced configuration
settings.

![Configuration Interface](/img/versioned_docs/threatprevention_7.4/threatprevention/reportingmodule/interface.webp)

Expand the Configuration menu in the Navigation pane for related pages:

- [Service Accounts Page](/docs/privilegesecure/4.1/configuration/services/service-accounts.md)
  — Add or modify service accounts
- Service Nodes:

  - [Service Nodes Page](/docs/privilegesecure/4.1/configuration/services/service-nodes.md)
    — View the status and details of Privilege Secure Services
  - [Scheduled Tasks Page](/docs/privilegesecure/4.1/configuration/scheduled-tasks.md)
    — View or modify recurring tasks

- System Settings — Modify the system settings:

  - [Action Service Settings Page](/docs/privilegesecure/4.1/configuration/services/action-service.md)
  - [Database Page](/docs/privilegesecure/4.1/configuration/database-configuration.md)
  - [Email Configuration Page](/docs/privilegesecure/4.1/configuration/email-configuration.md)
  - [Global Settings Page](/docs/privilegesecure/4.1/configuration/global-settings.md)
  - [Local Account Password Options Page](/docs/privilegesecure/4.1/configuration/password-options.md)
  - [Password History Options Page](/docs/privilegesecure/4.1/configuration/password-options.md)
  - [Local Account Password Options Page](/docs/privilegesecure/4.1/configuration/password-options.md)
  - [Services Page](/docs/privilegesecure/4.1/configuration/services/index.md)

- [Authentication Page](/docs/privilegesecure/4.1/configuration/authentication/index.md)
  — Add or modify multi-factor authentication (MFA)
- [Integration Connectors Page](/docs/privilegesecure/4.1/configuration/integrations/index.md)
  — Configure settings for integration with other applications
- SIEM:

  - [SIEM Server Page](/docs/privilegesecure/4.1/configuration/siem/siem-server.md)
    — Add or modify SIEM servers
  - [SIEM Templates Page](/docs/privilegesecure/4.1/configuration/siem/siem-templates.md)
    — Add or modify SIEM templates
