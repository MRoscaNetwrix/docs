# Session Timeout

For security reasons, the Privilege Secure Console automatically logs out the user after 20 minutes
of inactivity. A Session Timeout warning message appears after 15 minutes.

![Session Timeout window](/img/product_docs/privilegesecure/4.1/accessmanagement/enduser/sessiontimeout.webp)

If the timeout message appears, click Stay Logged In to continue using the console.
