# Dashboard Interface

The Dashboard interface displays an overview of activity sessions, users, resources and related
information.

![Dashboard Interface](/img/versioned_docs/privilegesecure_4.1/privilegesecure/accessmanagement/enduser/activedashboarduser.webp)

The overview section shows information for the following:

- Active Dashboard – Shows all currently active sessions. See the
  [Active Dashboard](/docs/privilegesecure/4.1/user-guide/dashboard/active-sessions.md)
  topic for additional information.
- Scheduled Dashboard – Shows all scheduled sessions. See the
  [Scheduled Dashboard](/docs/privilegesecure/4.1/user-guide/dashboard/scheduled-sessions.md)
  topic for additional information.
- Approvals Dashboard – Shows sessions waiting for approval. See the
  [Approvals Dashboard](/docs/privilegesecure/4.1/user-guide/dashboard/approvals.md)
  topic for additional information.
- Historical Dashboard – Shows previous sessions. See the
  [Historical Dashboard](/docs/privilegesecure/4.1/user-guide/dashboard/historical-sessions.md)
  topic for additional information.

The table shows information on the selected activity session.
