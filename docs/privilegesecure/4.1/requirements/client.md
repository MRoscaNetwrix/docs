# Client Requirements

Privilege Secure is a web service which can be accessed locally or remotely if the server’s firewall
permits it. The supported browsers for Privilege Secure are:

- Microsoft® Edge® Chromium
- Google® Chrome® 54.0 or later (Recommended)
- Apple® Safari®
- Mozilla® Firefox®

:::note
The browser compatibility mode must be turned off to access the Privilege Secure web
service.

:::
