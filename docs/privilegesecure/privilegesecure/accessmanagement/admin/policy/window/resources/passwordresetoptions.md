# Password Reset Options Window

The Password Reset option control password resets for the selected account.

Click the __Password Reset Options__ button to open the window and configure the following:

![passwordresetoptions](/static/img/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/policy/window/resources/passwordresetoptions.png)

Select options to rotate the account password when the session starts or ends.

When clicking __Okay__ button, the password reset policy is applied to an account.
