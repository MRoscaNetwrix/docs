# Configuration Interface

The Configuration interface provides information and management options for advanced configuration settings.

![Configuration Interface](/static/img/product_docs/threatprevention/threatprevention/reportingmodule/interface.png)

Expand the Configuration menu in the Navigation pane for related pages:

- [Service Accounts Page](/docs/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/configuration/page/serviceaccounts.md) — Add or modify service accounts
- Service Nodes:

  - [Service Nodes Page](/docs/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/configuration/page/servicenodes.md) — View the status and details of Privilege Secure Services
  - [Scheduled Tasks Page](/docs/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/configuration/page/scheduledtasks.md) — View or modify recurring tasks
- System Settings — Modify the system settings:

  - [Action Service Settings Page](/docs/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/configuration/page/actionservicesettings.md)
  - [Database Page](/docs/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/configuration/page/database.md)
  - [Email Configuration Page](/docs/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/configuration/page/emailconfiguration.md)
  - [Global Settings Page](/docs/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/configuration/page/globalsettings.md)
  - [Local Account Password Options Page](/docs/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/configuration/page/localaccountpasswordoptions.md)
  - [Password History Options Page](/docs/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/configuration/page/passwordhistoryoptions.md)
  - [Local Account Password Options Page](/docs/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/configuration/page/localaccountpasswordoptions.md)
  - [Services Page](/docs/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/configuration/page/services.md)
- [Authentication Page](/docs/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/configuration/page/authentication.md) — Add or modify multi-factor authentication (MFA)
- [Integration Connectors Page](/docs/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/configuration/page/integrationconnectors.md) — Configure settings for integration with other applications
- SIEM:

  - [SIEM Server Page](/docs/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/configuration/page/siemserver.md) — Add or modify SIEM servers
  - [SIEM Templates Page](/docs/product_docs/privilegesecure/privilegesecure/accessmanagement/admin/configuration/page/siemtemplates.md) — Add or modify SIEM templates
