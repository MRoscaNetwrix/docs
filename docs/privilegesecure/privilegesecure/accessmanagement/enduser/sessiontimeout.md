# Session Timeout

For security reasons, the Privilege Secure Console automatically logs out the user after 10 minutes of inactivity. A Session Timeout warning message displays after 5 minutes.![Session time out window](../../../../../static/img/product_docs/privilegesecure/privilegesecure/accessmanagement/enduser/sessiontimeout.webp)If the timeout message displays, click Stay Logged In to continue using the console.
