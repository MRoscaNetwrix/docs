# Changing a Single Review Status

The status of a Single Review and be changed from **Complete** to **Not Started**. An Administrator or UARÂ Admin role is required to make the status change.

1. Note the Name of the review to be changed. This example uses *0001\_Membership\_SOD Edit Role*.

   ![](../../../../../static/images/enus/Content/Resources/Images/uar_change_status1.png "Note the review name")
2. Open **Customization** > **List, Records, & Field** > **Record Types**
3. Locate **Review** in the list.

   ![](../../../../../static/images/enus/Content/Resources/Images/uar_change_status2.png "Locate Review in the Edit column")
4. Click **List** in the **Review** row.

   ![](../../../../../static/images/enus/Content/Resources/Images/uar_change_status3.png "List displayed for Review")
5. Click **Edit** in the row of the review to be changed. *0001\_Membership\_SOD Edit Role* for this example.

   ![](../../../../../static/images/enus/Content/Resources/Images/uar_change_status4.png "Edit the review")
6. Change **Status** from **Complete** to **Not Started**.
7. Click **Save**.
