# Example: Deployment or Library Script Changes related to an Approved Script Change

Use Case

A developer plans to make changes to a Suitelet, including the Suitelet library file and depoyment.

1. Developer creates a Change Request and attaches the Suitelet:

   ![](../../../../static/images/enus/Content/Resources/Images/opp_clearance_Case1-1.png)
2. IT Approvers approve the change.
3. Developer makes necessary changes to the script.

**Expected Result**

1. The change logs for the Suitelet, the Suitelet library script and deployment are attached to the related script approved Change Request.

   ![](../../../../static/images/enus/Content/Resources/Images/opp_clearance_Case1-2.png)
2. The logs are compliant.

   ![](../../../../static/images/enus/Content/Resources/Images/opp_clearance_Case1-3.png)
