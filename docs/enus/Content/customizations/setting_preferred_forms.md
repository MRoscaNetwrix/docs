# Setting Preferred Forms

You can set the preferred form to use for specific tasks.

1. Open **Customization** > **Lists, Records, & Fields** > **Record Types**
2. Select a form from the list. For this example, select **Change Request**.

   ![](../../../../static/images/enus/Content/Resources/Images/preferred_form1.png "Setting the preferred form.")
3. Open the **Forms** tab.

   ![](../../../../static/images/enus/Content/Resources/Images/preferred_form2.png "Setting the preferred form.")
4. Click the **Preferred** radio button to make the new **ITGC**Â Change Request form the preferred form.
5. Click **Save**.
