# Customizations API

The Customizations API provides external access to retrieve customization from a NetSuite account.

Customizations API command:

* [Get Customizations](get_customizations "Open Get Customizations topic")

Try the [Customizations](https://documenter.getpostman.com/view/30883336/2s9YeABubu "Try Get Customizations") API in Postman. The **Get Customizations** API is published to Postman, where you can try it out and test it.
