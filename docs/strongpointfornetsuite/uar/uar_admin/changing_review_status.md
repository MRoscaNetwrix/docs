# Changing a Single Review Status

The status of a Single Review and be changed from __Complete__ to __Not Started__. An Administrator or UARÂ Admin role is required to make the status change.

1. Note the Name of the review to be changed. This example uses _0001\_Membership\_SOD Edit Role_.

   ![Note the review name](/img/product_docs/strongpointfornetsuite/uar/uar_admin/uar_change_status1.png)
2. Open __Customization__ > __List, Records, & Field__ > __Record Types__
3. Locate __Review__ in the list.

   ![Locate Review in the Edit column](/img/product_docs/strongpointfornetsuite/uar/uar_admin/uar_change_status2.png)
4. Click __List__ in the __Review__ row.

   ![List displayed for Review](/img/product_docs/strongpointfornetsuite/uar/uar_admin/uar_change_status3.png)
5. Click __Edit__ in the row of the review to be changed. _0001\_Membership\_SOD Edit Role_ for this example.

   ![Edit the review](/img/product_docs/strongpointfornetsuite/uar/uar_admin/uar_change_status4.png)
6. Change __Status__ from __Complete__ to __Not Started__.
7. Click __Save__.
