# Setting Preferred Forms

You can set the preferred form to use for specific tasks.

1. Open __Customization__ > __Lists, Records, & Fields__ > __Record Types__
2. Select a form from the list. For this example, select __Change Request__.

   ![Setting the preferred form.](/img/product_docs/strongpointfornetsuite/customizations/preferred_form1.png)
3. Open the __Forms__ tab.

   ![Setting the preferred form.](/img/product_docs/strongpointfornetsuite/customizations/preferred_form2.png)
4. Click the __Preferred__ radio button to make the new __ITGC__Â Change Request form the preferred form.
5. Click __Save__.
