# Update Field Description and Help

Documentation only changes are always compliant. If [Opportunistic Clearance](../change_management/opportunistic_clearance.md) is on, this object change is reported in the Change Log as __Documentation Change__ for the __Change Type__ and the __Resolution__ set to _Automatically cleared documentation change. Only Help or Description changed_. The Change Log is closed.

1. Open __Strongpoint__ > __Clean Up__ > __Update Field Description and Help__You can filter the results or check __Description/Help is Empty__ to focus the __Field List__. Click on a column heading in the __Field List__ to sort the list by the selected column.

   ![cust_ui_help_update](../../../static/img/product_docs/strongpointfornetsuite/clean_up/cust_ui_help_update.webp)
2. Add or edit __Description__ and __Help__ text.
3. Click __Update__
