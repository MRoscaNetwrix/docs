# Installing Flashlight by Strongpoint

The Flashlight app package is available on the Salesforce AppExchange. To install Flashlight by Strongpoint:

1. Paste one of these links into a browser:  
   **Production org**: [login.salesforce.com/packaging/installPackage.apexp?p0=04tf4000003RQud](http://login.salesforce.com/packaging/installPackage.apexp?p0=04tf4000003RQud "Use this link for a Production Org")  
   **Sandbox org**: [https://test.salesforce.com/packaging/installPackage.apexp?p0=04tf4000003RQud](https://test.salesforce.com/packaging/installPackage.apexp?p0=04tf4000003RQud "Use this link for a Sandbox org")
2. Log in to Salesforce when prompted.
3. Select **Install for Admins Only**
4. Click **Install**
![](../../../../static/images/StrongpointSalesforceFlashlight/Content/Resources/Images/install_flashlight1_800x399.png "Install Flashlight by Strongpoint")

:::note
NOTE: Strongpoint installs in the background. An email notification is sent to you when the installation is complete.
:::
