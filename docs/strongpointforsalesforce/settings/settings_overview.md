# Settings Overview

The Settings menu is located on the Netwrix Dashboard page.

The Settings tab accesses:

- __Getting Started Wizard__ guides you through the installation process.
- __Configuration and Stats__ is a live report of your Automated Documentation system.
- __Platform Governor Status__ is a live status update of the Usage of SFDC Governor Limit.
- __Netwrix Knowledge Center__: opens the Netwrix Knowledge Center.
- __SPÂ Credentials/Test Connection__ sets up the credentials for the connected app. Required to run Scanners. Tests the connection to help troubleshoot connectivity issues.
- __Upgrade / Downgrade Package__ modifies your license.
