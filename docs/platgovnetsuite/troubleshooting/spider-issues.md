# Spider Page Not Found

If you see the **Spider Page Not Found** error:

1. Open **Setup** > **Company** > **General Preferences**
2. Make sure **Web Site Hosting Files Always Available** is checked.
3. Click **Save**

# Spider Spins and Does Not Finish

If you kick off a spider and the gears on the page spin for more than an hour, this is likely an
issue with the **IP Restriction**. Contact Netwrix support at
[https://www.netwrix.com/support.html ](https://www.netwrix.com/support.html) for assistance in
configuring your account.
