# Jobs

A job is defined via the ```Job``` tag to orchestrate tasks together, in order to perform specific actions.

All [task types](/versioned_docs/usercube_6.1/usercube/integration-guide/toolkit/xml-configuration/jobs/tasks/index.md) are child elements of jobs.
