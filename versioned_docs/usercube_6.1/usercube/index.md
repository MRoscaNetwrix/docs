# A software solution to match your IGA needs

To learn about Netwrix Usercube and build the solution you need, explore our guides.

The present documentation mentions the Netwrix Usercube application as simply Usercube.

__Usercube's guides__ include:

- An [Introduction Guide](/versioned_docs/usercube_6.1/usercube/introduction-guide/index.md) if you are new to Usercube.
- A [User Guide](/versioned_docs/usercube_6.1/usercube/user-guide/index.md) to configure Usercube from scratch via the UI.
- An [Integration Guide](/versioned_docs/usercube_6.1/usercube/integration-guide/index.md) to complete Usercube's configuration in XML according to your needs.
- An [Installation Guide](/versioned_docs/usercube_6.1/usercube/installation-guide/index.md) to install Usercube in a production environment.
- A [Migration Guide](/versioned_docs/usercube_6.1/usercube/migration-guide/index.md) to upgrade to a new version of Usercube.
- [Release Notes](/versioned_docs/usercube_6.1/usercube/whatsnew/index.md) to get details about specific changes in Usercube's updates.
