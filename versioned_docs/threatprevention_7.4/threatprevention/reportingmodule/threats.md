# Threats

Threat data is available in the full version of Netwrix Threat Manager.

![Threats interface](/img/versioned_docs/threatprevention_7.4/threatprevention/reportingmodule/threats.png)

The Threats interface in the Netwrix Threat Manager Reporting Module provides information about the
full feature. It includes:

- Launch a Product Tour – Opens the Welcome to the Netwrix products Guided Tour website page, that
  grants access to an online demo
- Request a quote – Opens the Discover the Pricing on Netwrix Products website page, where you can
  request pricing information
