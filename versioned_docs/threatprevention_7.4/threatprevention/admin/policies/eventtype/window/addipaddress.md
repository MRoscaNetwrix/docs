# Add IP Address Window

The Add IP Address window provides a textbox to enter the IP address to be included or excluded.

![Add IP Address Window](/img/versioned_docs/threatprevention_7.4/threatprevention/admin/policies/eventtype/window/addipaddress.png)

Entered IP addresses are added to the appropriate box on the IP Addresses (from) filter or the IP Addresses (to) filter, from where you clicked the __Add__ (+) button to open this window.
