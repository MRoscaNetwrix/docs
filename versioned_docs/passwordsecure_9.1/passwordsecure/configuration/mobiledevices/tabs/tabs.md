# Tabs

Once you have successfully logged in, you will find yourself in the view where all the user's passwords are located.

![all passwords in mobile app](/img/versioned_docs/passwordsecure_9.1/passwordsecure/configuration/mobiledevices/tabs/all-passwords-ma-en.png)

Here you have the following options:

Action menu

With a click on ![three-points-en](/img/versioned_docs/passwordsecure_9.1/passwordsecure/configuration/mobiledevices/tabs/three-points-en.png) the action menu is opened.

![actions mobile app](/img/versioned_docs/passwordsecure_9.1/passwordsecure/configuration/mobiledevices/tabs/actions-ma-en.png)

The following actions are offered:

- __Open settings__ (more information can be found in the Settings chapter).
- __Close tab__ (the option is offered only if you are in one of the organizational units tabs. The default ones are excluded)
- __Logout__ (you will be logged out from the database)
- __Cancel__ (closes the action menu and returns to the tab view)

Tabs

Below the passwords there is a bar for managing tabs.

![manage tabs](/img/versioned_docs/passwordsecure_9.1/passwordsecure/configuration/mobiledevices/tabs/all-passwords-ma-2-en.png)

By clicking on the plus sign there is a possibility to add more tabs.

![add tabs](/img/versioned_docs/passwordsecure_9.1/passwordsecure/configuration/mobiledevices/tabs/add-tabs-ma.png)

These tabs are organizational units that the user can see. By default, the tabs __"All passwords"__ and __"Personal"__ are stored.
