# Recycle Bin

This option allows you to view and permanently delete deleted passwords to which you are entitled.

## Procedure for deleting passwords

To put passwords into the recycle bin there are 2 possible procedures. Select the passwords you want to delete and click on __Move to bin (1)__ or right-click on the passwords and select __Move to bin(2)__.

![bin_2](/img/versioned_docs/passwordsecure_9.1/passwordsecure/configuration/advanced_view/clientmodule/passwords/bin_2.png)

You will then be asked if you actually want to perform this action.

![bin_3](/img/versioned_docs/passwordsecure_9.1/passwordsecure/configuration/advanced_view/clientmodule/passwords/bin_3.png)
