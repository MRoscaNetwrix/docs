# Agent Properties Window

On the Agents tab, the Edit button opens the agent’s Properties window, which contains the following tabs:

- [Active Directory Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/activedirectory.md) – AD Agent only
- [AD Users Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/adusers.md)
- [API Server Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/apiserver.md)
- [Archiving Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/archiving.md)
- [Additional Properties Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/additionalproperties.md)
- [Connection Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/connection.md)
- [Disk Quota Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/diskquota.md)
- [Dell CEE Options Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/dellceeoptions.md) – Activity Agent only
- [ DNS Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/dns.md)
- [Inactivity Alerts Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/inactivityalerts.md)
- [Linux Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/linux.md) – Linux Agent only
- [NetApp FPolicy Options Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/netappfpolicyoptions.md) – Activity Agent only
- [Network Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/network.md)
- [Network Proxy Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/networkproxy.md)
- [Nutanix Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/nutanix.md) – Activity Agent only
- [Panzura Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/panzura.md) – Activity Agent only
- [Qumulo Tab](/versioned_docs/activitymonitor_7.1/activitymonitor/admin/agents/properties/qumulo.md) – Activity Agent only

Select the desired agent and click __Edit__ to open the agent’s Properties window.
