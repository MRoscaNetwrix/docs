# Console Restoration

Follow the steps to restore the list of agents managed on the Activity Monitor Console.

__Step 1 窶点_ Restore ```Agents.ini``` file.

__Step 2 窶点_ Restore ```FileMonitor.lic``` file.

__Step 3 窶点_ Start the console.

__Step 4 窶点_ Update the passwords if the console is restored on a different workstation.

The Activity Monitor Console can now connect to deployed agents.
