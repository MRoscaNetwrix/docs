# Add Taxonomy

On this step, you are prompted to load predefined taxonomies.

![processing_settings_taxonomies](/img/versioned_docs/dataclassification_5.6.2/ndc/deployment/processing_settings_taxonomies.png)

Click the search bar and select one or several taxonomies you want to add. See
[Built-in Taxonomies Overview ](/versioned_docs/dataclassification_5.6.2/ndc/taxonomies/standalone_taxonomies.md)
for the full list of built-in taxonomies supported by Netwrix Data Classification.
