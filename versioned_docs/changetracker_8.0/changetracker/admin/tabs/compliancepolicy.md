# Policy Tab

Detail on current and previous compliance assessments can be seen here with details of failures where they exist.

![compliancepolicytab](/img/versioned_docs/changetracker_8.0/changetracker/admin/tabs/compliancepolicytab.png)

- __Timeline__– Click to see the detail for previous compliance assessments.
- __Group Performance__ – An overall comparison for all systems within the group, current and previous results. You can click on either column to see the Detail view of the selected report run.
- __Device Scores__ – Bars show the distribution of device count for each scoring band, providing focus on the most common failures, click on any bar to see the Detail view of the selected score range.
- __Options__ – Run a new report or view/export previous reports.
- __Failures Analysis__ – Headline details of failures with the highest incidence.
