The Summary page displays a summary of the configured query. It is a wizard page for all categories.

![System Info Data Collector Wizard Summary page](/img/versioned_docs/enterpriseauditor_11.6/enterpriseauditor/admin/datacollector/adinventory/summary.png)

Click __Finish__ to save configuration changes. If no changes were made, it is a best practice to click __Cancel__ to close the System Info Data Collector Wizard ensuring that no accidental clicks are saved.
