# View Query

Use the __View Query__ option to open the Host Discovery Queries pane.

![View Query option on Activities pane](/img/versioned_docs/enterpriseauditor_11.6/enterpriseauditor/admin/hostmanagement/actions/viewquery.png)

Click __View Query__ to go to the Host Discovery Queries pane. See the [Host Discovery Queries](/versioned_docs/enterpriseauditor_11.6/enterpriseauditor/admin/hostdiscovery/queries.md) topic for additional information.
