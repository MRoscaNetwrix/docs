# System Management Screen

The System Management screen is available from the left hand menu within Privilege Secure under the main Management menu item:

![Screen_Shot_2021-11-16_at_7.10.27_PM.png](/img/versioned_docs/privilegesecurefordiscovery_2.21/privilegesecure/discovery/admin/reporting/4413252324887_screen_shot_2021-11-16_at_7.10.27_pm.png)

![Screen_Shot_2021-11-12_at_1.03.25_PM.png](/img/versioned_docs/privilegesecurefordiscovery_2.21/privilegesecure/discovery/admin/reporting/4413252324887_screen_shot_2021-11-12_at_1.03.25_pm.png)

The screen provides a list of views that can be utilized to narrow the list of systems to be viewed

![Screen_Shot_2021-11-16_at_7.12.42_PM.png](/img/versioned_docs/privilegesecurefordiscovery_2.21/privilegesecure/discovery/admin/reporting/4413252324887_screen_shot_2021-11-16_at_7.12.42_pm.png)

This is especially helpful for determining a list of machines for followup. This could be errors that need resolution or new machines requiring being added to protect mode.

Further narrowing can be provided by the filtering (Domain, OU, Operating System and time based filters).

![Screen_Shot_2021-11-16_at_7.15.23_PM.png](/img/versioned_docs/privilegesecurefordiscovery_2.21/privilegesecure/discovery/admin/reporting/4413252324887_screen_shot_2021-11-16_at_7.15.23_pm.png)

The result can be exported in a number of differing formats.

The name links on the left hand side of the screen will directly take a user to the page for the system to update the configuration and inventory of the system as required.

![Screen_Shot_2021-11-16_at_7.22.02_PM.png](/img/versioned_docs/privilegesecurefordiscovery_2.21/privilegesecure/discovery/admin/reporting/4413252324887_screen_shot_2021-11-16_at_7.22.02_pm.png)

These shortcuts can also be bookmarked for easy access to the system required.
