# General Commands

You can use the following Management Shell cmdlets to perform tasks such as:

- [Get-Computer](/versioned_docs/groupid_11.0/groupid/managementshell/general/getcomputer.md):
  provides information about a computer object.
- [Get-ConnectedStoreInformation](/versioned_docs/groupid_11.0/groupid/managementshell/general/getconnectedstoreinformation.md):
  provides information about the connected identity store.
- [Get-ConnectedUser](/versioned_docs/groupid_11.0/groupid/managementshell/general/getconnecteduser.md):
  provides information about the connected user.
- [Get-GroupIdInformation](/versioned_docs/groupid_11.0/groupid/managementshell/general/getgroupidinformation.md):
  provides information about GroupID.
- [Get-ImanamiCommand](/versioned_docs/groupid_11.0/groupid/managementshell/general/getimanamicommand.md):
  provides basic information about GroupID Management Shell cmdlets.
- [Get-ReplicationStatus](/versioned_docs/groupid_11.0/groupid/managementshell/general/getreplicationstatus.md):
  provides the replication status of objects in an identity store.
- [Get-TombStoneObject](/versioned_docs/groupid_11.0/groupid/managementshell/general/gettombstoneobject.md):
  displays information about tombstone objects.
- [Invoke-Replication](/versioned_docs/groupid_11.0/groupid/managementshell/general/invokereplication.md):
  starts the replication process for all the identity stores or for a specific identity store.
- [New-Container](/versioned_docs/groupid_11.0/groupid/managementshell/general/newcontainer.md):
  creates a new organizational unit.
- [Remove-Container](/versioned_docs/groupid_11.0/groupid/managementshell/general/removecontainer.md):
  removes an empty organizational unit.
- [Restore-TombStoneObject](/versioned_docs/groupid_11.0/groupid/managementshell/general/restoretombstoneobject.md):
  restores tombstone objects from the directory.
- [Send-Notification](/versioned_docs/groupid_11.0/groupid/managementshell/general/sendnotification.md):
  sends notifications to a group or a user.
