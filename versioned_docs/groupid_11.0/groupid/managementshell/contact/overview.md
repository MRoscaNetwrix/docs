# Contact Commands

GroupID provides the following cmdlets to perform contact-related tasks, such as:

- [Get-Contact](/versioned_docs/groupid_11.0/groupid/managementshell/contact/getcontact.md):
  retrieves a contact that matches the given criteria.
- [New-Contact](/versioned_docs/groupid_11.0/groupid/managementshell/contact/newcontact.md): creates
  a new contact.
- [Remove-Contact](/versioned_docs/groupid_11.0/groupid/managementshell/contact/removecontact.md):
  removes a contact from the directory.
- [Set-Contact](/versioned_docs/groupid_11.0/groupid/managementshell/contact/setcontact.md):
  modifies a contact in the directory.
