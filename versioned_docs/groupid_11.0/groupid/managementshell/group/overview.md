# Group Commands

This section covers cmdlets for performing tasks related to managed and unmanaged groups.

- [Convert-Group](/versioned_docs/groupid_11.0/groupid/managementshell/group/convertgroup.md): converts an unmanaged group to a Smart Group.
- [Expire-Group](/versioned_docs/groupid_11.0/groupid/managementshell/group/expiregroup.md): expires a group temporarily.
- [Get-Group](/versioned_docs/groupid_11.0/groupid/managementshell/group/getgroup.md): retrieves groups from one or more containers.
- [Move-Group](/versioned_docs/groupid_11.0/groupid/managementshell/group/movegroup.md): moves a group to a different container in the same domain or in a different domain.
- [New-Group](/versioned_docs/groupid_11.0/groupid/managementshell/group/newgroup.md): creates an unmanaged group.
- [Remove-Group](/versioned_docs/groupid_11.0/groupid/managementshell/group/remove-group.md): deletes a managed group, unmanaged group, or Dynasty in the directory.
- [Renew-Group](/versioned_docs/groupid_11.0/groupid/managementshell/group/renewgroup.md): reactivates an expired group.
- [Set-Group](/versioned_docs/groupid_11.0/groupid/managementshell/group/setgroup.md): modifies an unmanaged group in the directory.
