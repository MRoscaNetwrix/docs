# Entitlement

GroupID Entitlement enables you to stay informed on the permissions assigned to objects residing on your Active Directory servers and SharePoint sites.

See the [Entitlement](/versioned_docs/groupid_11.0/groupid/admincenter/entitlement/overview.md) topic for detailed information on Entitlement.
