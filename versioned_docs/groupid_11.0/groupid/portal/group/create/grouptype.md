# Group Type page

Select the type of group you want to create and click __Next__. Options are:

- [Create a Static Group](/versioned_docs/groupid_11.0/groupid/portal/group/create/activedirectory/group.md#Create-a-Static-Group)
- [Create a Smart Group](/versioned_docs/groupid_11.0/groupid/portal/group/create/activedirectory/group.md#Create-a-Smart-Group)
- [Create a Password Expiry Group](/versioned_docs/groupid_11.0/groupid/portal/group/create/activedirectory/group.md#Create-a-Password-Expiry-Group) (not supported in Microsoft Entra ID)
- [Create a Dynasty using the Organizational/Geographical/Custom template](/versioned_docs/groupid_11.0/groupid/portal/group/dynasty/activedirectory/createdynasty.md#Create-a-Dynasty-using-the-OrganizationalGeographicalCustom-template)
- [Create a Dynasty using the Managerial template](/versioned_docs/groupid_11.0/groupid/portal/group/dynasty/activedirectory/createdynasty.md#Create-a-Dynasty-using-the-Managerial-template)
- [Create Teams](/versioned_docs/groupid_11.0/groupid/portal/group/teams/create.md) (for Microsoft Entra ID only)
