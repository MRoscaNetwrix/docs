# Create Teams

Using GroupID portal, you can create Team in the identity store.

What do you want to do?

- Create Teams

## Create Teams

1. In the GroupID portal, click the __Create New__ button and select __Team__.

   The __Create Group__ wizard opens to the __Group Type__ page.

   NOTE: Pages and fields on the Create Group wizard may vary, since the administrator can customize the wizard by adding or removing pages and fields.
2. On the [Group Type page](/versioned_docs/groupid_11.0/groupid/portal/group/create/grouptype.md), select the required group type and click __Next__.
3. On the General page, specify basic information about the group.
4. If you select Static Group, specify members for the group on the [Members page](/versioned_docs/groupid_11.0/groupid/portal/group/create/activedirectory/members.md).
5. If you select Smart Group or a Dynasty, review and modify the query for updating group membership on the [ Smart Group page](/versioned_docs/groupid_11.0/groupid/portal/group/create/activedirectory/smartgroup.md).
6. On the [Owners page](/versioned_docs/groupid_11.0/groupid/portal/group/create/activedirectory/owners.md), specify primary and additional owners for the group.
7. On the [Summary Page](/versioned_docs/groupid_11.0/groupid/portal/user/create/activedirectory/summary.md), review the settings and then click Finish to complete the wizard.
