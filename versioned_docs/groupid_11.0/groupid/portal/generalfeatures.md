## Directory Search

The GroupID portal provides a robust search feature that empowers users to efficiently manage various directory objects within their identity store. These objects include mailboxes, users, groups, and contacts. Once a search is performed, the results are displayed on the __Search Results__ page.

See the [Directory Search](/versioned_docs/groupid_11.0/groupid/portal/search/search.md) topic.

## Find Dialog Box

The Find dialog box enables you to search __User__, __Group__, and __Contact__ objects in the connected identity store. The object types available for search may vary, depending on the page you launch the __Find__ dialog box from.

See the [Find Dialog Box](/versioned_docs/groupid_11.0/groupid/portal/search/find.md) topic.

## Portal Settings

GroupID Portal offers flexible portal settings to customize the user experience and personalize the portal for each user.

See the [Portal Settings](/versioned_docs/groupid_11.0/groupid/portal/setting/portal.md) topic.

## User Account Settings

Users can manage their own profile information and perform various account-related actions through the User Settings.

See the [User Account Settings](/versioned_docs/groupid_11.0/groupid/portal/setting/user.md) topic.

## Toolbars

The portal provides toolbars with diverse options that users can use on the objects. These actions include editing properties, managing membership, viewing history, and more.

See the [Toolbar](/versioned_docs/groupid_11.0/groupid/portal/toolbar.md) topic.
