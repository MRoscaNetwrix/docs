# Exchange page

(of Create Mailbox wizard)

1. In the __Alias__ box, type an alias for the mailbox.
2. In the __Mailbox Store__ list, select a mailbox store to create the user's mailbox on.
3. The __Subscription List__ provides the subscriptions offered by Office 365 to a mailbox user. Select the check boxes for the subscriptions you want to assign to this mailbox.

   The subscription list is displayed when Office 365 is configured as the messaging provider for the identity store.
4. Click __Next__.
