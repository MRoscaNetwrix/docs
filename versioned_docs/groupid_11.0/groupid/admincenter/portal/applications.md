# GroupID Applications

- [GroupID Portal](/versioned_docs/groupid_11.0/groupid/admincenter/portal/overview.md)
- [Data Service](/versioned_docs/groupid_11.0/groupid/admincenter/service/dataservice/overview.md)

- [Mobile Service](/versioned_docs/groupid_11.0/groupid/admincenter/mobileservice/overview.md)
