# Contact APIs

GroupID provides the following APIs to perform contact-specific functions:

- [Create a Contact](/versioned_docs/groupid_11.0/groupid/api/contact/createcontact.md)
- [Delete a Contact](/versioned_docs/groupid_11.0/groupid/api/contact/deletecontact.md)
- [Delete Contacts](/versioned_docs/groupid_11.0/groupid/api/contact/deletecontacts.md)
- [Get a Contact](/versioned_docs/groupid_11.0/groupid/api/contact/getcontact.md)
- [Get Contacts](/versioned_docs/groupid_11.0/groupid/api/contact/getcontacts.md)
- [Update a Contact](/versioned_docs/groupid_11.0/groupid/api/contact/updatecontact.md)

NOTE: The contact object type is not supported in a Microsoft Entra ID based identity store.
