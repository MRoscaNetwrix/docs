# Session Timeout

For security reasons, the Privilege Secure Console automatically logs out the user after 20 minutes of inactivity. A Session Timeout warning message appears after 15 minutes.

![Session Timeout window](/img/versioned_docs/privilegesecure_4.1/privilegesecure/accessmanagement/enduser/sessiontimeout.png)

If the timeout message appears, click Stay Logged In to continue using the console.
