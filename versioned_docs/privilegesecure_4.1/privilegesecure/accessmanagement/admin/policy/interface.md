# Policy Interface

The Policy interface provides users with options for creating access policies, investigating activity sessions, onboarding and managing users, groups, resources, and credentials. This topic explains the interface features and how to use them.

![Admin Policy Interface](/img/versioned_docs/threatprevention_7.4/threatprevention/reportingmodule/interface.png)

  - [Password Complexity Page](/versioned_docs/privilegesecure_4.1/privilegesecure/accessmanagement/admin/policy/page/passwordcomplexity.md) — Configure the password complexity rules for the platform resources
  - [Schedule Policies Page](/versioned_docs/privilegesecure_4.1/privilegesecure/accessmanagement/admin/policy/page/schedulepolicies.md) — Add or modify schedules for tasks and policies
