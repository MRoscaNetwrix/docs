# Mailbox Commands

This section covers cmdlets for performing mailbox-specific tasks such as:

- [Get-Mailbox](/docs/groupid/11.0/groupid/managementshell/mailbox/getmailbox.md):
  retrieves a mailbox.
- [New-Mailbox](/docs/groupid/11.0/groupid/managementshell/mailbox/newmailbox.md): creates
  a new mailbox.
- [Remove-Mailbox](/docs/groupid/11.0/groupid/managementshell/mailbox/removemailbox.md):
  deletes a mailbox.
- [Set-Mailbox](/docs/groupid/11.0/groupid/managementshell/mailbox/setmailbox.md):
  modifies a mailbox.
