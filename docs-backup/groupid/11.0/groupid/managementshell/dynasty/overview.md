# Dynasty Commands

This section covers the following cmdlets for managing Dynasties.

- [New-Dynasty](/docs/groupid/11.0/groupid/managementshell/dynasty/newdynasty.md): creates
  a new Dynasty.
- [Set-Dynasty](/docs/groupid/11.0/groupid/managementshell/dynasty/setdynasty.md):
  modifies a Dynasty or its children.
