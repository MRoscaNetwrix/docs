# Messaging System

Directory Manager enables users to configure messaging systems to efficiently create or edit mail
enabled objects through a Synchronize job. Users can sync or deprovision subscriptions from the
following messaging systems:

- [Exchange Subscription](/docs/directorymanager/11.1/directorymanager/portal/synchronize/job/exchange.md)
- [Google Workspace Subscription](/docs/directorymanager/11.1/directorymanager/portal/synchronize/job/googleapp.md)
- [Office 365 Subscription](/docs/directorymanager/11.1/directorymanager/portal/synchronize/job/office365.md)
