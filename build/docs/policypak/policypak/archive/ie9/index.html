Endpoint Policy Manager: Manage Internet Explorer (IE9) Using Group Policy

Internet Explorer's many settings, and users' tendency to want to tweak then to their taste, can be a challenge to network administrators. WithNetwrix Endpoint Policy Manager (formerly PolicyPak) you can leverage its pre-configured pak for Internet Explorer 9, which makes configuring Internet Explorer as simple as can be. Check out this video to see how it's done.
