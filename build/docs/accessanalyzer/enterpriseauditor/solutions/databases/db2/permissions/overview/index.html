Permissions Job Group

This job group provides insight into all types of permissions at the database and object level across all the targeted Db2 database servers.

The jobs in the Permission job group are:

  • Db2_DirectPermissions Job – Provides insight into direct user and role permissions to all the database objects in the targeted Db2 database servers
  • Db2_EffectivePermissions Job – Provides insight into effective user and role permissions to all the database objects in the targeted Db2 database servers
